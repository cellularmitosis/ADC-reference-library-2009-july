PowerStep

PowerStep is a feature of the DV SE model that allows the user to change the processor's clock speed, either slowing down (to 366 MHz) to conserve power or speeding up when the full 466-MHz speed is needed. The user interface for this feature is a check box labeled Reduce Processor Speed in the Advanced Settings panel of the Energy Saver control panel.
