Administrative Tasks

Most directory paths specified in the following tasks are relative to the installation directory, NEXT_ROOT. The definition of NEXT_ROOT depends on the platform you are using:
PlatformInstallation Directory
Mach/
HPUXDefined at installation time.
SolarisDefined at installation time.
Windows NTDefined at installation time.

Some tasks are special to WebObjects Enterprise and not WebObjects Pro. In addition, some tasks do not apply to WebObjects installed on Mach. These exceptions are noted in the task procedures.
