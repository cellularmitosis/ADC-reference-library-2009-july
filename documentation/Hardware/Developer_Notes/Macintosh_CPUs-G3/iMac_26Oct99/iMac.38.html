Sound System

The 16-bit stereo audio circuitry provides high-quality sound input and output through the built-in microphone and speakers. The user can also connect external input and output devices by way of the sound input and output jacks .

Note

The iMac family computers also support speakers and microphones that connect to the USB port.

The sound system is based on the Screamer codec IC and an output amplifier. The sound system supports sample sizes up to 16 bits at a sample rate of 44.1 kHz.
