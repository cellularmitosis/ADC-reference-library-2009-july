Technical Q&A HW48
Booting after invoking the Open Firmware user interface

Q: How do you boot after you are finished using the Open Firmware user interface?

A: Models prior to the iMacs display a startup message stating you can boot using the "boot" or "bye" commands. iMacs and later may have this message. At present, not all do. In this case use "boot" or "mac-boot".

[Feb 15 1999]
