Technical Q&A QA1245
Project Builder 2.1 needs projects to be checked into CVS to enable CVS support

Q: Project Builder 2.1, in the December 2002 Mac OS X Developer Tools, doesn't seem to allow me to use any CVS functionality, even though my source files are indeed checked into CVS. This seemed to work just fine in prior versions of the developer tools - what's wrong?

A: This is a known issue with Project Builder that should be fixed in the next major release of the Mac OS X Developer Tools after the December 2002 Mac OS X Developer Tools. The workaround is to check your Project Builder project (the .pbproj file) itself into CVS, at which point Project Builder will recognize CVS functionality.

[Mar 20, 2003]
