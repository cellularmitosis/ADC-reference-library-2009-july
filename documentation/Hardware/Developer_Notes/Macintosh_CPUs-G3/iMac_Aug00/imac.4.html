New Features

The features that have changed are listed here with links to the sections that describe them. For a comparison of new and old features, see Table 1-1 .

  • Processor speed: The microprocessor clock speed is 350, 400, 450, or 500 MHz. For more information, see G3 Microprocessor .

  • Memory: The RAM slots can support up to 512 MB each for a maximum total of 1 GB. For more information, see RAM Expansion Modules .

  • Hard disk storage: The capacity of the built-in hard disk drive is 7, 10, 20, or 30 GB. For more information, see Hard Disk Drive .

  • Target Disk Mode: The computer can act like a FireWire storage device connected to another computer. See Target Disk Mode .

  • Graphics acceleration: The IC used in the video circuits is an ATI RAGE 128 PRO 4XL. For more information, see Video Display Subsystem .

  • Keyboard: The computer comes with an Apple Pro Keyboard, a full-size USB keyboard. For more information, see Keyboard .

  • Mouse: The computer comes with an Apple Pro Mouse, a USB mouse with optical tracking. For more information, see Mouse .
