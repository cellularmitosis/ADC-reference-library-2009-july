Leopard Reference Library: Hardware & Drivers
Security
Apple provides opportunities for hardware developers to create devices, such as Smart Cards and fingerprint-verification devices, that work with Mac OS X software to provide authentication, authorization, or cryptographic services. Software developers can use security services to build security into their applications or enable performance of privileged operations.
