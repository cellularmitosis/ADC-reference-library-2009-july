Introduction to Hardware Developer Note Terms and Abbreviations

Contents:

Organization of This Document
See Also


This document provides a list of common terms and abbreviations used in Developer Notes that document Apple's hardware products and related technologies.

Apple's hardware product Developer Notes each provide an overview of the features and the main logic board architecture of a particular Mac computer model.

Apple's technology Developer Notes each provide an overview of a specific technology, such as AirPort, FireWire, and USB, and describe the implementation details of that technology specific to particular Mac computer models.

Developers who are designing peripheral devices or other products to be compatible with Mac computer will find this document useful.

Organization of This Document

This document contains the following sections:

  • “Units of Measure”

  • “Common Terms, Acronyms, and Abbreviations”

See Also

For specific information about a particular Mac computer, see the product Developer Note for that computer in the Guides > Hardware & Drivers > Apple Hardware category of the ADC Reference Library.

Apple offers the following technology Developer Notes, each providing an overview of the technology and details of its implementation on specific Mac computer models:

  • AirPort Developer Note

  • Audio Developer Note

  • Bluetooth Developer Note

  • Ethernet Developer Note

  • FireWire Developer Note

  • PCI Developer Note

  • RAM Expansion Developer Note

  • Universal Serial Bus Developer Note

  • Video Developer Note
