API Changes

The changes from a decompressor's (indeed, the entire sound chain's) perspective include support for extended forms of the SoundComponentData and SoundParamBlock records. Additionally, a client can now pass this information to the sound channel by using an extended form of the ScheduledSoundHeader with the scheduledSoundCmd.
