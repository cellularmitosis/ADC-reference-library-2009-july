Repetitions

A repetition (WORepetition) repeats its contents a specific number of times, much like a loop in a structured programming language. Typically, a repetition displays repeated data using items in a list, multiple rows in a table, or multiple tables.

A repetition contains two main attributes:

  • The list is the array of objects to display.
  • The item is the current object being displayed. Generally, you bind item to a variable and then use that variable in the repetition to stand for the current item.

To create a repetition that does not include a table row:

  1. Click the WORepetition button and add items to the repetition.
  2. Drag from the array that contains your data to the box beside the List icon .

    This binds the list attribute to they array
  3. Drag from the key for the current item to the box beside the Item icon .

    This binds the item attribute to the key.

When you create a repetition around a table row, the repetition icon doesn't appear on your page, so you must bind its attributes differently. Instead of displaying an icon, the row is shaded in blue. To create a repetition around a table row:

  1. Select an item in the row, click the <TR> tag in the path view, and click the WORepetition button .

    Note that selecting every cell in the row does not select the table row (<TR>) element, but only the individual table cell (<TD>) elements. To select the row, you must click on the <TR> tag in the path view.

    Tip: When you're creating a table, you can wrap one of its rows in a repetition automatically. In the New Table dialog, select "Second row is wrapped in a WORepetition."

  2. Drag from the array that contains your data to the <WORepetition> tag in the path view, and choose list from the menu that appears.
  3. Drag from the key that contains the current item to the <WORepetition> tag in the path view, and choose item from the menu that appears.
