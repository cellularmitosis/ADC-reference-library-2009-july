Class Description

VectorDataSource is a concrete subclass of DataSource that can be used to provide enterprise objects to a DisplayGroup without having to fetch them from the database. In a VectorDataSource, objects are maintained in an in-memory next.util.ImmutableVector.

VectorDataSource can fetch, insert, and delete objects. It can also provide a detail data source.

public next.eo.VectorDataSource(next.util.Coder aCoder)

Creates and returns a VectorDataSource object.

If classDescription and editingContext are specified, classDescription contains information about the objects provided by the VectorDataSource and editingContext is the VectorDataSource's editing context.

If aCoder is provided, the new VectorDataSource is decoded from aCoder. This constructor, together with encodeWithCoder, provide the basis for archiving and distributing qualifiers. aCoder is specified when a VectorDataSource is being read from an archive file; you never specify this constructor directly. For more information, see the next.util.Coding interface specification in the Foundation Framework Reference.
