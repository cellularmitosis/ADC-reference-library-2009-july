Leopard Reference Library: QuickTime
Games
The QuickTime API provides a high-level, cross-platform tool set for incorporating video, sound, graphics, text, interactivity, and music into games. Interactive QuickTime movies can also provide a lightweight cross-platform package for delivering small games. Depending on your application, any part of the QuickTime API may be useful in developing games. QuickTime can also be combined with other graphics technologies used in games, such as Quartz and OpenGL.
