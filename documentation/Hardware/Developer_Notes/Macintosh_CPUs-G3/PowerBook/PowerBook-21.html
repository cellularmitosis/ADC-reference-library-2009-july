Memory Controller and Bus Bridge

The Uni-N memory controller and bus bridge IC provides cost and performance benefits by combining several functions into a single IC. It contains the memory controller, the PCI bus bridge, the Ethernet and FireWire interfaces, and the AGP port.

In addition to the four buses listed in Table 2-1, the Uni-N IC also has separate interfaces to the physical layer (PHY) ICs for Ethernet and FireWire and an I 2 C interface that is used for configuring the memory subsystem.

Table 2-1  Buses supported by the Uni-N IC
