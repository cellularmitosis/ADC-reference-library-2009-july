Description

OutputBins2PDE demonstrates how to write a Cocoa PDE that supports all of the features of Mac OS X 10.5, while also running on Mac OS X 10.4. It demonstrates how to write a complex Cocoa Printer PDE, demonstrating the following features that are new in Mac OS X 10.5: resizing the printer dialog, overriding the Help and OK buttons, and automatic PPD constraint handling. When running on Mac OS X 10.4 the PDE automatically falls back to a reduced feature set.
