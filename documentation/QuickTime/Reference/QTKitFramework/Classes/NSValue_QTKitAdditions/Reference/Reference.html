Overview

The QuickTime Kit supports categories in the Foundation frameworkâ€™s NSValue class that allow you to get QTTime and QTTimeRange structures as objects of type NSValue. In Mac OS X v10.5, QTKit defines extra operations on the SMPTETime type. SMPTETime is defined in CoreAudio/CoreAudioTypes.h.
