HTML Editing Toolbar

Version: 1.0

Posted: 2007-06-11

Build Requirements: None, the sample is all HTML and JavaScript.

Runtime Requirements: Safari 3 or other WebKit browser on Mac OS X 10.5

View Source Code:

Download Sample (“FancyEditingToolbar.zip”, 22.5K)
Download Sample (“FancyEditingToolbar.dmg”, 28.7K)


Description

Implements an HTML editing toolbar for Safari using JavaScript and CSS that fades in and out and has buttons to apply text formating and text alignment. Click in the text area to see the toolbar appear.
