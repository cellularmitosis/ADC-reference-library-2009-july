New Features

Here is a list of the features that are new to the Power Mac G4 computer.

  • Microprocessor clock speed The clock frequency in the single-processor configurations is either 800 or 933 MHz. The dual-processor configuration has a clock speed of 1 GHz. For more information, see "PowerPC G4 Microprocessor".
  • RAM The standard configurations have either 256 or 512 MB of RAM installed. For more information, see "RAM Expansion".
  • Graphics card Depending on the confguration, the graphics card has either 32, 64, or 128 MB of display RAM. All the cards support two monitors. For more information, see "Graphics Card".
