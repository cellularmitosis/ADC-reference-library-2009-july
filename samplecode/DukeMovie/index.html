DukeMovie

Version: 1.2

Posted: 2006-05-11

Build Requirements: XCode 2.2, or Java 2 SDK for Windows, and QuickTime 7

Runtime Requirements: Java 1.5 and QuickTime 7, or later, recommended

View Source Code:

Download Sample (“DukeMovie.dmg”, 314.6K)


Description

Demonstrates QuickTime playback. Displays a window with a tumbling Duke animation above a Movie Controller. The file used for playback can be any type that QuickTime supports. Both the movie controller and window size will automatically adjust based on the type of media selected.

The window also contains controls allowing the user to open a new file, toggle looping, hide the media file and start sequential media types such as movies and audio files independant of the normal Quicktime Movie Controller.
