Leopard Reference Library: QuickTime
Streaming
The streaming API in QuickTime allows developers to recognize and play streaming movies, add hint tracks so movies can be streamed, create packetizers and reassemblers, mix streaming and nonstreaming data in a single movie, and broadcast live streams in real time. QuickTime Streaming Server modules provide an easy way for developers to add new functionality to the QuickTime Streaming Server.
