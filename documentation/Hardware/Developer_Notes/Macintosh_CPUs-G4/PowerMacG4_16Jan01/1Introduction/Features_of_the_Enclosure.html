Features of the Enclosure

The Power Mac G4 computer's enclosure is a mini-tower design with opaque side panels and transparent handles.

The front of the computer's enclosure has the media slots for the optical media drive and the optional Zip drive, three buttons—power, reset, and NMI—and the power-on light.

The back panel includes the A/C power socket, the I/O ports, and the openings for PCI cards.

The user can get access to the main logic board to install PCI cards or additional memory by swinging the door down.

The lower part of the enclosure has space for additional storage devices. See "Fixed-Media Drives".
