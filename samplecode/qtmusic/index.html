Description

QTMusic is a sample application that demonstrates ways of using the QuickTime Music Architecture to play notes and tunes. It is based on sample code written by David Van Brink and explained in develop issue 23. The ability to use custom instruments to play tunes and in movies. Can be compiled and run under the Mac OS and under Windows. 68k or PowerPC (or newer) machines running System 8.5 or later. PC with Intel Pentium or compatible processor. QuickTime 4.0 or later. CodeWarrior Pro 2 or later or Visual C++ 5.0 or later.
