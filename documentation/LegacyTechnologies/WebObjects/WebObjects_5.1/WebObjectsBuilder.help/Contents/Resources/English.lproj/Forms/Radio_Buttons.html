Radio Buttons

A group of radio buttons lets the user choose among several options. When one radio button in a group is selected, the others are deselected.

To add a radio button group to a form:

  1. Click the Radio Button button to add all the radio buttons.
  2. For every button in the group:
    • Bind the name attribute to the group's name.
    • Bind the selection attribute to the variable's name.
    • Bind the value attribute to the string to set the variable to when the user selects that button.
