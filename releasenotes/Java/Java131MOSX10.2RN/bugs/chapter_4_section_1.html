Known and Resolved Issues

This chapter lists both known issues and issues resolved between Java 1.3.1 Update 1 and Mac OS X 10.2. If you still have problems with any of the issues, please file a bug, as described in Filing and Tracking Bugs.

Each issue is broken down as follows:

  • A Radar number that Apple is using to track that issue is followed by a brief statement of the issue.

  • Description provides more details about the issue.

  • The Workaround field explains what Apple has done in these individual cases. For example, many times this field says, “This has been fixed,” meaning that the source code has been modified to correct this issue. In some cases, the Workaround field gives you additional information about the fix or may advise you on steps you should take in implementing your code.

For information on general changes and new features see What’s New for Java in Mac OS X Version 10.2.
