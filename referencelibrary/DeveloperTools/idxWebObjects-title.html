WebObjects
WebObjects and Xcode Tools provide a suite of tools for creating and deploying applications and web services for Mac OS X. Developers can use Xcode, Apple's integrated development environment, for WebObjects development and Interface Builder to add an Aqua interface to their WebObjects application. WebObjects also includes the EOModeler application, which developers can use to reverse-engineer existing databases and to design schemas for new databases.
