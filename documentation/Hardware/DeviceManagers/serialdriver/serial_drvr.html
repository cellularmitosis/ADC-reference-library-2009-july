Inside Macintosh: Devices
(Chapter 7 - Serial Driver)
This book describes how to write software that interacts with built-in and peripheral hardware devices. This book provides useful information for writing and debugging low-level software. Chapter 7 of Inside Macintosh: Devices describes how you can use the Serial Driver to transfer data to a device connected to a Macintosh modem or printer port.
