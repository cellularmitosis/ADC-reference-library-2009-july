Buttons

Buttons let you submit or cancel a form. The Submit button submits the user's data back to the server. The Reset button returns the form to its initial state without invoking an HTML request. The Image button is similar to a submit button but displays an image instead of text.

To add a Submit button to a form:

  • Click the Submit Button button , and bind the action attribute to an action method.

To add a Reset button to a form:

  • Click the Reset Button button .

To change the text displayed in a Submit or Reset button:

  • Set the value attribute to the text.

To add an Image button:

  1. Click the Image Button button , and bind the action attribute to an action method.
  2. Set the image's filename attribute.

    If the filename is known now, bind filename to it.

    If the filename will be known when WebObjects generates the page, bind filename to a key that contains the name.

To add more than one Submit button or Image button to a form:

  • In the form's dynamic Inspector, set the multipleSubmit attribute to true.
