The Queue Utilities provide routines that let you manipulate queue data structures maintained by the Mac OS. You can also use the Queue Utilities to create and manipulate your own queues.

Inside Macintosh: Operating System Utilities
(Chapter 6 - Queue Utilities)
This document describes the parts of the Macintosh system software that allow you to manage various low-level aspects of the operating system. Chapter 6 describes how you can add elements to and delete them from an operating-system queue. It also describes how you can create and manipulate your own queues.
