Hyperlinks

WebObjects Builder lets you create two types of hyperlinks:

  • A static hyperlink (<A HREF="...">) has a destination that doesn't change.
  • A dynamic hyperlink (WOHyperlink) has a destination that's determined when WebObjects generates the page.

To create a static hyperlink:

  1. Choose Elements > Hyperlink or click the Static Hyperlink button , and type the text for the browser to display.

    Tip: You can also select the text the browser displays and then press the Static Hyperlink button.

  2. In the Inspector, set the hyperlink's destination.

    You can also choose whether the destination is a relative or absolute reference.

Tip: While the static link's destination cannot change, you can change the text the browser displays by using a dynamic string.

To create a dynamic hyperlink:

  1. Choose WebObjects > WOHyperlink or click the Dynamic Hyperlink button , and type the text for the browser to display.

    Tip: You can also select the text the browser displays and then press the Dynamic Hyperlink button.

  2. Set the hyperlink's destination.

    There are three ways to do this:
    • To link to another page in your application, bind the pageName attribute to the page's name.
    • To perform an action, bind the action attribute to the action method's name.
    • To link to a specific URL, bind the href attribute to the URL.

To switch a hyperlink between static and dynamic:

  • In the Inspector, click the Make Static or Make Dynamic button.

To change a static hyperlink to an anchor:

  • In the Inspector, deselect "Link to another document," select "Named location (anchor) in this document," and enter the anchor's name.
