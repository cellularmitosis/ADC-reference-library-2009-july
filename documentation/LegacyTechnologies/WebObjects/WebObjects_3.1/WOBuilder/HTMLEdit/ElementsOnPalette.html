HTML Elements on the Static Elements Palette

You can create any of these HTML tags using icons on the Static Elements palette:

  • <A> (See "Creating Hyperlinks")
  • <ADDRESS>
  • <BLOCKQUOTE>
  • <H1> through <H6>
  • <HR>
  • <OL>
  • <P>
  • <PRE>
  • <TABLE> (See "Creating Tables")
  • <UL>
