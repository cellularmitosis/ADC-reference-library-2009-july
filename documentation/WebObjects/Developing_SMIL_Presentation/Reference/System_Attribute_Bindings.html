System-Attribute Bindings

Table 4-18 lists the bindings you can use with elements inside WOSMILSeq elements. Your multimedia player uses the values of these bindings to determine which element of a group of elements inside a WOSMILSwitch gets displayed.


Table 4-18 System-Attribute bindings


Binding Description
systemBitrate The system's bandwidth.
systemCaptions Specifies whether the user has opted to see closed-captioning information. Value: "on" or "off".
systemLanguage The languages the user has chosen.
systemOverdubOrCaption Specifies whether the user has chosen caption or overdub. Value: "caption" or "overdub".
systemRequired The name of an extension.
systemScreenSize The minimum screen resolution required.
systemScreenDepth The minimum color depth.
