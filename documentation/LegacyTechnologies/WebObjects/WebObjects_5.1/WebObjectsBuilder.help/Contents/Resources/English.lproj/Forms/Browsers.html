Browsers

A browser lets the user choose among several items. The user can choose more than one item.

To add a browser to a form:

  • Click the Browser button , and then bind the menu's list, item, and selection attributes.
    • list is the array of objects to choose from.
    • item is a temporary variable. As WebObjects iterates through list to display its items, it holds the item it's currently displaying in the variable that item specifies. You can use that variable in other bindings to stand for the item being displayed.
    • selections is an array of the items the user has selected.

By default, the browser uses item's toString method to generate the names it lists. To choose a different method or instance variable to generate the names:

  • Bind the menu's displayString attribute to the new value.

To choose whether a user can choose more than one item in the browser:

  • In the browser's dynamic Inspector, set the multiple attribute.

    If it's true, the user can select more than one item. If it's false, the user can select only one item.

    Note that even if multiple is false, the selections variable is set to an array of one item.
