Running the Script

To use the script, you drag a movie file, or multiple files, or folders of movie files onto the droplet. A dialog ( Figure 31 ) appears prompting you for the destination folder for the finished files.

Figure 31 Specifying the destination folder for finished files

Once chosen, a media skin copy of each dragged-on movie is created and placed into this folder.
