Leopard Reference Library: Mac OS X Server
Streaming
Mac OS X Server's QuickTime streaming API allows developers to recognize and play streaming movies, add hit tracks so that movies can be streamed, create packetizers and reassemblers, mix streaming and nonstreaming data in a single movie, and broadcast live streams in real time. Developers can also easily add new functionality to the QuickTime Streaming Server modules. Administrators can use the Server Admin application to administer a QuickTime Streaming Server and QTSS Publisher to manage media and prepare it for streaming or progressive download.
