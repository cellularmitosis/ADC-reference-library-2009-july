WCSTOL_L(3)              BSD Library Functions Manual              WCSTOL_L(3)

NAME
     wcstoimax_l, wcstol_l, wcstoll_l, wcstoul_l, wcstoull_l, wcstoumax_l -- convert a wide character string
     value to a long, unsigned long, long long, unsigned long long, intmax_t or uintmax_t integer

LIBRARY
     Standard C Library (libc, -lc)

SYNOPSIS
     #include <wchar.h>
     #include <xlocale.h>

     long
     wcstol_l(const wchar_t * restrict nptr, wchar_t ** restrict endptr, int base, locale_t loc);

     long long
     wcstoll_l(const wchar_t * restrict nptr, wchar_t ** restrict endptr, int base, locale_t loc);

     unsigned long
     wcstoul_l(const wchar_t * restrict nptr, wchar_t ** restrict endptr, int base, locale_t loc);

     unsigned long long
     wcstoull_l(const wchar_t * restrict nptr, wchar_t ** restrict endptr, int base, locale_t loc);

     #include <inttypes.h>
     #include <xlocale.h>

     intmax_t
     wcstoimax_l(const wchar_t * restrict nptr, wchar_t ** restrict endptr, int base, locale_t loc);

     uintmax_t
     wcstoumax_l(const wchar_t * restrict nptr, wchar_t ** restrict endptr, int base, locale_t loc);

DESCRIPTION
     The wcstol_l(), wcstoul_l(), wcstoll_l(), wcstoull_l(), wcstoimax_l() and wcstoumax_l() functions are
     extended locale versions of the wcstol(), wcstoul(), wcstoll(), wcstoull(), wcstoimax() and wcstoumax()
     functions, respectively.  Refer to their manual pages for details.  Also, see xlocale(3) for more
     information about extended locales.

SEE ALSO
     wcstol(3), xlocale(3)
