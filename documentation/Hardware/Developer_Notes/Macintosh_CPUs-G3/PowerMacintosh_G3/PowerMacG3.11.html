Machine Identification and NewWorld Software

The Power Macintosh G3 described in this note and other Macintosh computers that implement the NewWorld software architecture no longer have unique machine ID values. Applications cannot determine which computer they are running on by reading the Gestalt Manager gestaltMachineType values. If an application requires a name string value for the computer model, the application should instead check the name registry for the value of the "Devices:device-tree:model" property to get a name string for the machine. The current model property string is " PowerMac1,1 ".
