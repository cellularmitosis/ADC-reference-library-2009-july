QuickTime Atoms

QuickTime Atoms are a highly flexible and modular format for creating extensible data structures. QuickTime Atom-based data structures are widely used in QuickTime. The QuickTime API includes a full suite of functions for manipulating data stored in QuickTime Atoms.

  • For an introduction to Atoms, QT Atoms, and AtomContainers, see Introduction to Atoms.

  • For complete documentation of the functions used to create and manipulate atoms, see Movie Toolbox: Data Types.

  • For code samples showing how these functions are used in QuickTime movies, see Creating, Copying, and Disposing of Atom Containers.
