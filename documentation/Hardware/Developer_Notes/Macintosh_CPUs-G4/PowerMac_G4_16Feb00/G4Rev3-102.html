ROM-in-RAM Architecture

The system software in all current Macintosh computers uses a ROM-in-RAM approach, also called the New World architecture, as described in Software For more information about this architecture, see Technote 1167, NewWorld Architecture, available on Apple's technote website at

http://developer.apple.com/technotes/tn/tn1167.html
