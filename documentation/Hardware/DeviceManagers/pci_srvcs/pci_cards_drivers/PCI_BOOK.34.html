Video Drivers

To present a seamless transition between the Open Firmware user interface and the user interface used by the Mac OS, the NewWorld architecture provides a mechanism for communication of the display mode, resolution, and so on, between the Open Firmware video driver and the Mac OS video driver. See New Name Registry Functionality for additional details.
