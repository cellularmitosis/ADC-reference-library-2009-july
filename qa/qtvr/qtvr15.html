Technical Q&A QTVR15
Launching a PICT from a Panorama

Q What do I need to do to correctly launch a PICT from a QTVR Panorama?

A Objects and PICTs can only be launched from a panorama from within Director or Hypercard. The QTVRPlayer will not do this for you. You need to write the code in Director or Hypercard to perform this task. (Samples of this code are contained in the testbed that comes with the QTVR Tool Suite.)

You should also make sure the name of the hotspot is the path name to the file relative to the projector/stack.

[Sep 01 1995]
