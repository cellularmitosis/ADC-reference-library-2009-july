Security Features

Here are the key security features supported by the system software on the Xserve computer.

  • Secure ports Xserve prevents mounting of CDs as well as hot-plugged USB and FireWire hard drives by means of an enclosure lock.
  • Secure remote management Xserve's remote monitoring and management tools run over encrypted links.
