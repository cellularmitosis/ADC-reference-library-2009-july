Technical Q&A CS11
Universality of ColorSync Accelerators and CMMs

Q: Is the ColorSync accelerator a standard which will accelerate all ColorSync CMM's or is it specific to a particular third-party CMM?

A: The ColorSync team defined a generic acceleration layer that all CMMs can take advantage of. Apple's default CMM calls the acceleration component if it is available. Third-party CMMs should take advantage of this generic acceleration layer as well.

[Mar 05 1996]
