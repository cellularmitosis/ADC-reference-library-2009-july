Description

Developers looking for the latest version of this sample should download the Core Audio SDK 1.4.2+.

This sample shows how to write a basic filter audio unit with a custom Cocoa user interface. The code demonstrates how to handle user presets, and includes a custom property for communication between the audio unit and its view used to update the response curve in the user interface.
