Manager Group: Common OS Services


Carbon supports the commonly-used managers in this group that aren't heavily dependent on operating-system implementation. For example, because Apple events offer a more abstract means of communicating between processes on both Mac OS 8 and 9 and on Mac OS X than does the PPC Toolbox on Mac OS 8 and 9, the PPC Toolbox is not supported by Carbon.
