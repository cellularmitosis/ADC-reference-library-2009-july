EIDE Interface

The KeyLargo IC provides an EIDE interface (ATA bus) that supports the the DVD-DOM drive and the wireless LAN module. The DVD-ROM drive is an ATAPI drive and is connected as device 0 (master) in an ATA Device 0/1 configuration.
