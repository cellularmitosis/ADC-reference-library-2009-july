Technical Q&A PLAT21
Determining Power PC Type 11 Errors

Q: How can I determine if a type 11 error on a PowerPC machine is a hardware error or a software error?

A: Install Macsbug. A software error will be correctly caught by MacsBug and reported as the appropriate software error condition. A hardware error will be reported as a type 11 error, since that means "miscellaneous hardware exception error."
