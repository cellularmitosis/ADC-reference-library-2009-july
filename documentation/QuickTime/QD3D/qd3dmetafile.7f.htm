End Group Objects

Labels

ASCII
EndGroup
Binary
endg ( = 0x656E6467 )

Data Format

No data.

Data Size

0

Description

An end group object is placed immediately after the last object in a group and is used to delimit that group.

Parent Hierarchy

3DMF.

Parent Objects

None.

Child Objects

None.

Example

BeginGroup ( DisplayGroup ( ) )             # empty group
