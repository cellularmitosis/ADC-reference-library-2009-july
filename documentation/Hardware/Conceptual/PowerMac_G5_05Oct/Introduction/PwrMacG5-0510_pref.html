Introduction to Power Mac G5 Developer Note

Contents:

Organization of This Document
See Also


This note gives a technical description of the 64-bit Power Mac G5 introduced in October 2005. It provides information about the computer’s internal design, input-output features, and expansion capabilities to help developers design products compatible with this computer.

Organization of This Document

This document provides a functional block diagram and descriptions of distinguishing features and main logic board components of the Power Mac G5.

See Also

For information about using Apple's hardware documentation, see Getting Started with Hardware and Drivers.

The “Features” section contains cross references to technology developer notes that provide more information about technologies supported by Power Mac G5 computers.

Apple offers the following additional resources for learning about the Power Mac G5:

  • Apple's Hardware site provides feature descriptions and technical specifications for currently shipping Macintosh computers.

  • Apple's Support site provides downloadable owner manuals, links to support services, and tips for getting the most out of using Macintosh computers.

  • Apple's Specifications site provides technical specifications for Macintosh computers.

  • Hardware Developer Note Terms and Abbreviations lists the standard units and abbreviations used in this developer note.
