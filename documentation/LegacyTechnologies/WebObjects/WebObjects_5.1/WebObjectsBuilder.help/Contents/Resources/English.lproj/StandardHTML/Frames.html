Frames

To create a frameset, use the File > New Frameset command. Edit the frameset with the special buttons in its tool bar. In a frame's Inspector, you can set the frame's name, whether the user can resize it, whether it contains scroll bars, and whether it's bordered.

To create a new frameset:

  • Choose File > New Frameset.

To add a frame to a frameset:

  • Select an existing frame and click the Split Vertically or Split Horizontally button.

To choose the page that a frame displays when the frameset is first loaded:

  • Click in the frame and press the Connect button .

To set a frame's name:

  • In the item's Inspector, use the Name field.

To choose how the frames in a particular row or column are sized:

  • Use the size buttons on the toolbar:
    • To size a row or column using a specific point size, click the Absolute Height or Absolute Width button.
    • To size a row column using a percentage, click the Percentage Height or Percentage Width button.
    • To size a row or column using the rest of the available space, use the Proportional Height or Proportional Width button.

To set the sizes for the frames:

  • Drag the borders between the frames.

    Note that each frame displays its size.

When you want a hyperlink to display its link in a specific frame, the hyperlink must specify the frame's name in that link. To set the frame's name:

  • In the hyperlink's Inspector, use the Frame field.
