Support for Text Drawing

There is now support for text drawing APIs on QDGraphics. This gives the capability to use some minimal but useful text drawing services from QuickTime on both platforms. There a few methods that have been added to the QDGraphics class that support simple text rendering. The SDK contains an example of the usage of these API services.

There are miscellaneous additions to the ComponentDescription class to return more information about loaded components. Additions to the set and get info calls on the SPBDevice class in the quicktime.sound package, with provision of support for device selection in the SGSoundChannel of the sequence grabbing services.
