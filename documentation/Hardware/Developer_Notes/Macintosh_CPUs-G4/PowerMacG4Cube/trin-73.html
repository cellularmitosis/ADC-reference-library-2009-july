RAM Expansion Modules

The RAM expansion modules for the Power Mac G4 Cube computer are 168-pin SDRAM DIMMs that are 3.3 volt, unbuffered, 8-byte, non-parity, and PC-100 compliant. The speed of the SDRAM devices must be rated at 125 MHz (8 ns) or faster.

IMPORTANT

RAM expansion DIMMs for the Power Mac G4 Cube must be PC-100 compliant and use SDRAM devices. If the user installs only DIMMs that use EDO or SGRAM devices, the computer will flash the power indicator several times when the user attempts to restart the computer. If the user mixes different types of DIMMs, the computer will use only the PC-100 SDRAM devices and ignore the others.
