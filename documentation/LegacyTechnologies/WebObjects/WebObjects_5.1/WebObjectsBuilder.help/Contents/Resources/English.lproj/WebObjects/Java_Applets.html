Java Applets

You can add Java applets to your component with the WOApplet class.

To create an applet that's already in your project:

  • Click in the toolbar and set its code attribute in the Inspector.

To create an applet that's not in your project:

  • Drag a file of type .class into your component.

Tip: You can also drag an element from the Client-Side Components palette to your component (see Using Palettes).
