Manager Group: Application Utilities


The application utilities consist of facilities for extending human interface features and data-sharing capabilities beyond those provided by the Human Interface Toolbox. Of these facilities, the Apple Guide Manager is replaced in Carbon by Apple Help, whereas Carbon offers no support or current replacements for the Control Strip Manager or the Data Access Manager. The other application utilities are either Carbon-compatible or are fully supported in Carbon.
