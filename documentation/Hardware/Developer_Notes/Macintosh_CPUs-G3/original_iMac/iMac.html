Apple iMac Computer

The Apple iMac computer is a new consumer-oriented desktop computer. This developer note is a technical description of the iMac, with the
emphasis on the features that are new or different from those of other Macintosh desktop computers.

The information in this note is arranged in five chapters.

  • Introduction introduces the iMac computer and describes its features.
  • Architecture describes the internal logic of the computer, primarily the main ICs.
  • Devices and Ports describes the I/O ports and the built-in I/O devices.
  • RAM Expansion provides development guides for the main RAM and video RAM expansion modules.
  • Software describes the system software that comes with the computer, with emphasis on the new Open Firmware features.

Additional information is in Abbreviations and in the section
Supplemental Reference Documents.
