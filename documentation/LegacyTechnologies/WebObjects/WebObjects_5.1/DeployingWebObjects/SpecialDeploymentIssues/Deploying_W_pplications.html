Deploying WebObjects 4.5.1 and WebObjects 5 Applications

If you want to deploy WebObjects 4.5.1 and WebObjects 5 or WebObjects 5.1 applications on the same computer, you must

  • install WebObjects 4.5.1 Deployment and either WebObjects 5 or WebObjects 5.1 Deployment on the computer
  • use either the WebObjects 5, or 5.1 versions of Monitor and wotaskd

Note that you cannot use the 4.5.1 version of wotaskd together with the 5 or 5.1 version of Monitor. Furthermore, you cannot deploy 4.5.1, 5, and 5.1 applications on the same computer.
