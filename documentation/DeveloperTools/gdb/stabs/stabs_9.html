Table of Symbol Descriptors

The symbol descriptor is the character which follows the colon in many stabs, and which tells what kind of stab it is. See section The String Field, for more information about their use.

digit
(
-
Variable on the stack; see section Automatic Variables Allocated on the Stack.
:
C++ nested symbol; see See section Defining a Symbol Within Another Type.
a
Parameter passed by reference in register; see section Passing Parameters by Reference.
b
Based variable; see section Fortran Based Variables.
c
Constant; see section Constants.
C
Conformant array bound (Pascal, maybe other languages); section Passing Conformant Array Parameters. Name of a caught exception (GNU C++). These can be distinguished because the latter uses N_CATCH and the former uses another symbol type.
d
Floating point register variable; see section Register Variables.
D
Parameter in floating point register; see section Passing Parameters in Registers.
f
File scope function; see section Procedures.
F
Global function; see section Procedures.
G
Global variable; see section Global Variables.
i
See section Passing Parameters in Registers.
I
Internal (nested) procedure; see section Nested Procedures.
J
Internal (nested) function; see section Nested Procedures.
L
Label name (documented by AIX, no further information known).
m
Module; see section Procedures.
p
Argument list parameter; see section Parameters.
pP
See section Parameters.
pF
Fortran Function parameter; see section Parameters.
P
Unfortunately, three separate meanings have been independently invented for this symbol descriptor. At least the GNU and Sun uses can be distinguished by the symbol type. Global Procedure (AIX) (symbol type used unknown); see section Procedures. Register parameter (GNU) (symbol type N_PSYM); see section Parameters. Prototype of function referenced by this file (Sun acc) (symbol type N_FUN).
Q
Static Procedure; see section Procedures.
R
Register parameter; see section Passing Parameters in Registers.
r
Register variable; see section Register Variables.
S
File scope variable; see section Static Variables.
s
Local variable (OS9000).
t
Type name; see section Giving a Type a Name.
T
Enumeration, structure, or union tag; see section Giving a Type a Name.
v
Parameter passed by reference; see section Passing Parameters by Reference.
V
Procedure scope static variable; see section Static Variables.
x
Conformant array; see section Passing Conformant Array Parameters.
X
Function return variable; see section Parameters.
