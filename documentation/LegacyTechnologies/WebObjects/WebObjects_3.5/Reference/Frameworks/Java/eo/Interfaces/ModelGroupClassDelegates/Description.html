ModelGroupClassDelegates

Interface Description

The ModelGroupClassDelegates interface defines a method that the ModelGroup class (not instances) can invoke in its delegate. Delegates are not required to provide implementations for all of the methods in an interface. When you write a delegate, you don't have to use the implements keyword to specify that the object implements the ModelGroupClassDelegates interface. Instead, simply use the ModelGroup setClassDelegate static method to assign your object as the delegate and then declare and implement the ModelGroupClassDelegates method.

Instance Methods

defaultModelGroup

public abstract next.eo.ModelGroup defaultModelGroup()

If implemented by the class delegate, this method should return the ModelGroup to be returned in response to the message defaultGroup. If this delegate method returns null, ModelGroup uses the default behavior of the defaultGroup class method.
