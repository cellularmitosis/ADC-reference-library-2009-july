Guidelines for Developers

Each expansion bay module must be designed to prevent damage to itself and to the computer when the user inserts or removes an expansion bay module with the computer running.

The expansion bay connector is designed so that when the module is inserted the ground and power pins make contact before the signal lines.

Even though you can design an expansion bay module that minimizes the possibility of damage when it is inserted hot--that is, while the computer is running--your instructions to the user should include warnings about the possibility of data corruption.
