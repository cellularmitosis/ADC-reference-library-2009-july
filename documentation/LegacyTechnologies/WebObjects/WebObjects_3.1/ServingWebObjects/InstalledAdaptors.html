Installed HTTP Adaptors

When WebObjects is installed, the following adaptors are put in NEXT_ROOT/NextLibrary/WOAdaptors along with source code. Note that only the CGI adaptor is installed on Mach, since neither Netscape's or Microsoft's servers have been ported to this platform. Also, no ISAPI binary file is installed on Solaris or HPUX platforms (only source code).
