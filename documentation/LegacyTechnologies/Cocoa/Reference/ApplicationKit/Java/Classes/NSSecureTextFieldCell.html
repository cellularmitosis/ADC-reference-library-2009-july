NSSecureTextFieldCell

Inherits from
NSTextFieldCell : NSActionCell : NSCell : NSObject
Implements
NSCoding (NSCell)
Package
com.apple.cocoa.application
Companion guide
Text Fields

Overview

NSSecureTextFieldCell works with NSSecureTextField to provide a text field whose value is guarded from user examination. It overrides the general cell use of the field editor to provide its own field editor, which doesn’t display text or allow the user to cut, copy, or paste its value.

Sets whether the receiver echoes bullets for each character typed.
