Manager: AltiVec Libraries


The AltiVec Libraries are compatible with Carbon on both the Mac OS X and Mac OS 9 platforms. However, these functions are currently not exported by the Carbon libraries. To use the AltiVec Libraries, link to vecLib under 9 from your Carbon application and it will work on OS X as well.
