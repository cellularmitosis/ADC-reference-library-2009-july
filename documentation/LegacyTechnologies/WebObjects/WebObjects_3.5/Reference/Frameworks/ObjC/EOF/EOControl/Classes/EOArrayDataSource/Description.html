Class Description

EOArrayDataSource is a concrete subclass of EODataSource that can be used to provide enterprise objects to an EODisplayGroup without having to fetch them from the database. In an EOArrayDataSource, objects are maintained in an in-memory NSArray.

EOArrayDataSource can fetch, insert, and delete objects. It can also provide a detail data source.
