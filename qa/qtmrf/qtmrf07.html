Technical Q&A QTMRF07
Linked Movies with Different Color Palettes

Q Can a QuickTime movie contain "subMovies" that have different color palettes?

A No. If a QuickTime movie references data from movie files that have their own 8-bit color palette, only the first color palette will be used in playing back all the files.

[Feb 15 1996]
