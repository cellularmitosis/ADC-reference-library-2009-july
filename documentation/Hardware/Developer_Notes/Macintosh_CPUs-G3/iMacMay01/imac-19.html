System RAM

The memory subsystem in the iMac supports two slots for 168-pin DIMMs (dual inline memory modules). The data bus to the RAM and DIMM is 64 bits wide, and the memory interface is synchronized to the 60x bus interface at 100 MHz. See RAM Expansion.
