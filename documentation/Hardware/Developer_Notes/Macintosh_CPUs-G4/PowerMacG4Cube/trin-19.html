System RAM

The memory subsystem in the Power Mac G4 Cube supports three slots for 168-pin DIMMs (dual inline memory modules). The data bus to the RAM and DIMM is 64 bits wide, and the memory interface is synchronized to the MaxBus interface at 100 MHz. For more information, see RAM Expansion.
