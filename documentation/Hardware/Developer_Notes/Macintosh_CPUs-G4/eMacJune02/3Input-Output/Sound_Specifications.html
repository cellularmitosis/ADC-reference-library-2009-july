Sound Specifications

The frequency response of the sound circuits, not including the microphone and speakers, is within plus or minus 1 dB from 20 Hz to 20 kHz.

Total harmonic distortion plus noise (THD+N) as a percentage of full scale are shown in Table 3-7.


Table 3-7 Distortion specifications

Connector THD+N
Line input 0.03%
Headphone jack, open circuit 0.03%
Headphone jack, 32-ohm headphones 0.5%

The signal-to-noise ratios (SNR) for various inputs and outputs are shown in Table 3-8. The values shown are unweighted.


Table 3-8 SNR specifications

Connector Signal-to-noise ratio
Microphone 65 dB
Headphone jack 90 dB
Line input 90 dB
Internal CD or DVD 90 dB
