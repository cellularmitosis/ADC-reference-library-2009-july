Context Menus

The elements in the edit window and in the path view have context menus that perform various operations on the element or its content. To access these menus, Control-click the element in Rhapsody or right-click the element in Windows NT.

Objects in the object browser also have context menus; to access them, Control-click the object in Rhapsody or right-click the object in Windows NT.
