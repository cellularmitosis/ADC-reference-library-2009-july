Paragraphs

Click to create a new paragraph. If there is a text selection, the entire selection becomes a paragraph.

You can use the Inspector to set the paragraph to one of the following tags:

  • Plain (<P>)

  • Preformatted (<PRE>)

  • Address (<ADDRESS>)

  • Block quote (<BLOCKQUOTE>)

  • Division (<DIV>)
