Manager: JManager


JManager is not supported in Carbon. On Mac OS X, the JManager has been replaced by the Java Embedding API contained in the JavaEmbedding framework. This new Java Embedding API consists mostly of applet embedding and utility functions. You can get most of the functionality previously provided by the JManager through the JNI Invocation API or the Core Foundation API on Mac OS X.
