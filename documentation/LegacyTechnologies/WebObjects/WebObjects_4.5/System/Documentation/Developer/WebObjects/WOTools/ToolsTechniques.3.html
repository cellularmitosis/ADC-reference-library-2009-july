Choosing Assistance

If your application doesn't access a database, choose None for Available Assistance.

If your application accesses a database, you also choose None if you want to develop the entire application yourself. However, you may wish to use one of the levels of assistance that WebObjects provides. Information about the Direct to Web option can be found in the Direct To Web chapter. Information about the Database Wizard can be found in "Creating a WebObjects Database Application" in Getting Started With WebObjects . Information about the Java Client option can be found in Creating a Java Client WebObjects Application .
