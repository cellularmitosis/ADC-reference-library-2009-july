Address Multiplexing

Signals A[0] - A[13] on each RAM SO-DIMM make up a 14-bit multiplexed address bus that can support several different types of SDRAM devices. Table 4-7 lists the types of devices that can be used in this computer by size, configuration, and sizes of row and column addresses.

IMPORTANT

The PowerBook G3 Series computer supports only the types of SDRAM devices specified in Table 4-7 . Other types of DRAM devices should not be used with this computer.
