64-bit File Offset Support for Mac OS 9

QuickTime 4.1 now includes support for 64-bit file offsets in Mac OS 9, which allow the use of files greater than 2 gigabytes (2G). QuickTime 4 had already been enhanced to include comprehensive support for 64-bit file offsets on Windows.

For more information, refer to Letters from the Ice Floe, Dispatch 22 at

<http://www.apple.com/quicktime/developers/icefloe/dispatch022.html>.
