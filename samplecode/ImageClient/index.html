Description

This sample shows how to use:

CFNetwork via synchronous and asynchronous operations.

CFNetDiagnositcs to help your application handle networking problems.

CFNetwork with system-provided proxy settings.

Authentication:

- simple authentication (Basic and digest)

- session authentication with persistence

- proxy authentication

- KeyChain usage
