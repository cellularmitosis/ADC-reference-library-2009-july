Carbon
Carbon is a set of C APIs that can be used to create Mac OS X Server applications that take advantage of features like protected memory, preemptive multitasking and dynamic resource allocation, and the Aqua user interface. Because Carbon is a collection of C programming interfaces, developers can use virtually any Macintosh C development environment that supports PowerPC to build Carbon applications.
