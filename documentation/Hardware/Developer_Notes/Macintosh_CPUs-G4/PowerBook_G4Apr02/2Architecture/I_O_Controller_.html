I/O Controller

The I/O controller IC in the PowerBook G4 computer is a custom IC called KeyLargo. It provides the interface and control signals for the following devices and functions:


Note: In the device tree, the I/O controller is named "mac-io".

"DMA Support"
"Interrupt Support"
"USB Interface"
"Ultra ATA/66 Interface"
"EIDE Interface"
"Modem Support"
"Sound Circuitry"
"Power Controller"
"AirPort Card Interface"
"CardBus Controller IC"
