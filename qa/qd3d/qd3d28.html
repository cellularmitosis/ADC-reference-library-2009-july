Technical Q&A QD3D28
QuickDraw 3D Documentation

Q: How can I get some good pointers on the more intricate workings of QuickDraw 3D?

A: As an Apple Partner or Associate Plus member you are encouraged to send your questions to feedback.

Another good source for up-to-date information is our World Wide Web site at http://product.info.apple.com/qd3d/QD3D.HTML

Apple's develop magazine has several good articles with QD3D information:

  • "QuickDraw 3D: A new Dimension for Macintosh Graphics", develop 22, p. 6
  • "The Basics of QuickDraw Graphics", develop 23, p. 30
  • "Graphical Truffles: Making the most of QuickDraw 3D", develop 24, p. 42
  • and more coming in future issues, including articles on custom attributes, debugging, and an introduction to NURB Curves.

develop articles can be downloaded via the web from http://developer.apple.com/dev/techsupport/develop/index.html

The monthly Developer CD Series also has QD3D information, especially the Tool Chest and Reference Library editions. Use the Find File command and search these CDs for filenames containing the string "3D".

All QuickDraw 3D programmers should have the manual 3D Graphics Programming with QuickDraw 3D (Addison Wesley), available from the Apple Developer Catalog. This is essentially the Inside Macintosh volume for QuickDraw 3D.

The CD-ROM that accompanies the book contains files with much more information than that in the printed pages. The Adobe Acrobat version of the manual, "3D Graphic Prog w/QD3D" is a 1,226 page superset of the printed manual. The Toolbox Assistant (QuickView) database file, "QuickDraw 3D.qv", provides quickly accessible, on-line documentation for all QuickDraw 3D data structures.

[Feb 09 1996]
