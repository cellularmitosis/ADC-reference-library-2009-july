Leopard Reference: Games
Files & Software Installation
In Mac OS X, software such as games, frameworks, and plug-ins is packaged in the form of bundles, which are directories in the file system that group related resources. It is often possible for users to simply drag such software to their hard disks from a CD, mounted disk image, or other location. For games with more complex installation requirements, developers can create installation packages and let users install them with installation tools. Executable bundles can support dynamic configuration through mechanisms such as property lists, user preferences, and environment variables.
