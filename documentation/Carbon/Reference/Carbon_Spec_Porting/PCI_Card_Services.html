Manager: PCI Card Services


Carbon does not support most PCI card services functions. In Mac OS X, all code that communicates directly with PCI cards must use the IOKit API.

However, Carbon does support many utility functions in the DriverServices.h and DriverSynchronization .h headers (such as time conversion and atomic memory functions) which have little or nothing to do with drivers. Several functions are useful, for example, to perform multiprocessing operations. In the future, these APIs may be moved into different header files.
