Bitblitz

Version: 1.0

Posted: 2003-10-10

Build Requirements: N/A

Runtime Requirements: Carbon Color QuickDraw

View Source Code:

Download Sample (“Bitblitz.zip”, 92.9K)
Download Sample (“Bitblitz.dmg”, 169.3K)


Description

The tool was designed to be a learning/testing tool for the Copybits(), CopyMask(), and CopyDeepMask() QuickDraw calls. It makes extensive use of GWorlds as well as the above mentioned QuickDraw bit tools. Requirements: Color QuickDraw Keywords: Bitblitz, CopyBits, CopyMask, CopyDeepMask, GWorld, NewGWorld, Graphics
