Modem Support

The KeyLargo IC has a Macintosh serial port that is the interface to the modem connector. The KeyLargo IC provides DB-DMA support for the modem interface. The modem provides analog call progress signals to the Screamer sound IC.

The internal hardware modem is a separate module that contains the modem ICs (controller and datapump) and the interface to the telephone line (DAA). For more information about the modem, see Internal Modem .
