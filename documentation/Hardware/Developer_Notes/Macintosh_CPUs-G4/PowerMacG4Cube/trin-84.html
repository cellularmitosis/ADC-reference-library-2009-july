Power Manager

The Power Manager has been redesigned to reduce power consumption in Sleep mode. Power Manager 2.0 is a native Mac OS manager designed to implement common power management policy across all Macintosh models by means of the new Power Plugin component.

Although the previous public Power Manager interfaces will be maintained for application compatibility, you should use the new API in your applications. Information about the API changes for Power Manager 2.0 is provided by Technote 1190, which can be obtained from the website at

http://developer.apple.com/technotes/tn/tn1190.html
