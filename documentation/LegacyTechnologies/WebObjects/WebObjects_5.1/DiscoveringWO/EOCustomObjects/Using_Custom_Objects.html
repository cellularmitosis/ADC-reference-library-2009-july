Using Custom Objects

You've already seen how easy it is to manipulate database rows by representing them as EOGenericRecords. Now, instead of using generic enterprise objects, you'll create an enterprise object class and customize its behavior.

In this chapter, you

  • generate a custom Java class
  • add custom logic to an enterprise object class
  • learn how to set default values for enterprise objects's properties
