Description

A simple Image Capture application which demonstrates preferred techniques for working with the Image Capture APIs. The application is built in a series of eight different steps. Each step is a self-contained project which adds new functionality to the application. The final project is the full-featured application MyPhoto.

Here's a brief list of the techniques demonstrated in this sample:

1) Finding out if / what Image Capture device is connected

2) Registering for device notification

3) Getting device properties

4) More device properties

5) Getting image properties

6) Downloading images

7) Controlling a digital camera

8) Using ImageIO to get and display image thumbnails, displaying images.
