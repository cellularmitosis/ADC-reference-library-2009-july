OSEndianTypes.h

Includes:
<libkern/OSTypes.h>
<libkern/OSByteOrder.h>

Overview

C++ inline types for byte swapping

Discussion

The OSEndianTypes consist of a number of types that are used very similarly to the traditional MacOS C scalar integers types, eg. UInt32 and SInt32.
