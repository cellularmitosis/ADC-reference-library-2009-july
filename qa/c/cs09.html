Technical Q&A CS09
Using Embedded EPS Profiles

Q: Can ColorSync 2.0 profiles be embedded in EPS and then used?

A: Yes, you can embed ColorSync profiles into EPS, as well as PICT and TIFF formats. For specific details on how to embed profiles, as well as sample code, see Advanced Color Imaging for the Mac OS, found on the Reference Library CDs, in the technical documentation/Inside Macintosh folder.
