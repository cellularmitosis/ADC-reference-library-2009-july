ModelGroupDelegates

Class Description

The ModelGroupDelegates interface defines methods that a ModelGroup can invoke in its delegate. Delegates are not required to provide implementations for all of the methods in the interface. When you write a delegate, you don't have to use the implements keyword to specify that the object implements the ModelGroupDelegates interface. Instead, simply use the ModelGroup setDelegate method to assign your object as the delegate and then declare and implement any subset of the methods declared in ModelGroupDelegates. The ModelGroup can determine if the delegate doesn't implement a delegate method and only attempts to invoke the methods the delegate actually implements.

Instance Methods

entityRelationshipForRow

public abstract next.eo.Relationship entityRelationshipForRow(next.eo.Entity anEntity, next.util.ImmutableHashtable row, next.eo.Relationship aRelationship)

Invoked when relationships are instantiated for a newly fetched object. The delegate can use the information in row to determine which entity the target enterprise object should be associated with, and replace the relationship appropriately.

modelGroupEntityWithName

public abstract next.eo.Model modelGroupEntityWithName(next.eo.ModelGroup group, java.lang.String name)

If implemented by the delegate, this method should search the group for the entity named name and return the entity's Model. Return null if name is not an entity in group.

relationshipFailedToLookupDestinationWithName

public abstract next.eo.Entity relationshipFailedToLookupDestinationWithName(next.eo.Relationship relationship, java.lang.String entityName)

Invoked when loading relationship and the destination entityName specified in the model file cannot be found in the model group. This most often occurs when a model references entities in another model file that can't be found. If the delegate doesn't implement this method, an exception is thrown. If the delegate does implement this method, the method's return value is set as the destination entity. if the delegate returns null, the destination entity is set to null.
