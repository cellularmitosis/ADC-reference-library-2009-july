Miscellaneous User Space API Reference

Various headers in /usr/include contain HeaderDoc markup. This API reference is a compendium of that content.

Many of these headers are third-party APIs with comments in non-HeaderDoc formats that closely resemble HeaderDoc. These comments are presented as-is, and formatting for these APIs may differ as a result.
