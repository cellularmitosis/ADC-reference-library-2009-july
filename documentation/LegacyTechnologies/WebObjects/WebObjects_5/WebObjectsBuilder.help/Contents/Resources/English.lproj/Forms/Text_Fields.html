Text Fields

A text field lets the user enter a line of text.

To add a text field to a form:

  • Click the Text Field button , and then bind the field's value attribute to the variable that holds the user's data.

To format the field's contents automatically:

  • In the item's dynamic Inspector, enter a format string in the dateformat or numberformat attribute.

    You can type a format string, or click the downward pointing arrow at the far right end of the field and choose a format string.

To choose whether the field's contents are visible:

  • In the Static Inspector, choose one of the following:
    • "Normal text input field" displays exactly what the user types.
    • "Password field" displays bullets instead what the user types.
    • "Hidden text field" is not displayed at all and lets you submit data that you do not want the user to see.

To set the field's length:

  • Use the "Visible length" and "Maximum Length" fields.

    "Visible length" is the size in characters of the text field's box. "Maximum length" is the maximum number of characters the user can enter.
