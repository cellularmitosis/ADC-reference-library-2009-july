Manager Group: Core Foundation


Core Foundation abstracts several operating system utilities fundamental to Mac OS 8 and 9 and to Mac OS X. For example, the CFString services assist you in manipulating Unicode strings -- an important facility for creating applications easily adaptable for worldwide markets. Core Foundation also supports a plug-in architecture, a preferences-settings mechanism, code and data sharing, and other highly useful features for your application. Carbon fully supports all Core Foundation services.
