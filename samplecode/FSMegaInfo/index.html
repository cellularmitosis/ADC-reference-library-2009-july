Description

This tool is designed to help folks who are implementing a plug-in file system (VFS plug-ins) on Mac OS X. You can use it to print information about various file system objects (volumes, directories, and files). This allows you to compare the information on your volume with the information returned by canonical Apple file systems, such as HFS Plus, AppleShare, and MS-DOS/FAT.
