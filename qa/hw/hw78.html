Technical Q&A HW78
PCI Header types

Q: Can you explain the three types of PCI headers that each PCI device must provide in its configuration header space?

A: Yes, there are types 0, 1, and 2. A type 0 header defines a PCI device such as display, Ethernet, mass storage, SCSI, etc. A type 1 header defines PCI to PCI bridges. Finally, type 2 headers define CardBus bridges of host type PCI.

[Nov 15 1999]
