Java Graphics Issues

Radar # 2756936

The printed page's Graphics2D is clipped incorrectly.

Description:

When printed, the page is offset and incorrectly clipped for the printer.

Resolution:

Software changed.


Radar # 2789524

AWT crashes accessing ColorSync information.

Description:

Random crashes in AWT, usually when starting up. AWT shouldn't have more than one thread in Carbon at a time, and ColorSync uses Carbon, so AWT's usage of ColorSync needs to mind the CarbonLock.

Resolution:

Software changed.
