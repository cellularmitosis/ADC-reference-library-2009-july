Aperture
Aperture is an application for performing professional postproduction processing and maintenance of RAW, JPEG, and TIFF images. It preserves the metadata and pixel integrity of the original image file throughout the postproduction workflow. Developers can use the Aperture API to control the export of images from Aperture.
