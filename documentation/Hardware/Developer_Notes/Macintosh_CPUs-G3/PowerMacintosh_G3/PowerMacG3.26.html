Display Memory

The display memory on the accelerated graphics card is separate from the main memory. The display memory consists of 16 MB of 100 MHz/10 ns SDRAM devices configured to make a 64-bit data bus. The display memory cannot be expanded.

With the 16 MB of video memory, the display data generated by the computer can have pixel depths of 8, 16, or 32 bits for monitors up to1920 by 1080 pixels. For a complete listing of the display resolutions, see Display Resolution Modes .
