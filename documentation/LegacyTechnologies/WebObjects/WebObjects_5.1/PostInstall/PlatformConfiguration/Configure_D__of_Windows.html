Configure Default Character Encoding on the Japanese Version of Windows

If you have installed WebObjects on the Japanese Version of Windows, you need to configure the default character encoding. The default character encoding is determined by the first character in NEXT_ROOT\Library\Frameworks\Foundation.framework\Resources\CStringEncoding. If this file exists, make sure that the first character is the number "8". If the file does not exist create it.
