Technical Q&A QA1185
Why don't all of my PDE localizations show up in all applications?

Q: Why do my PDE localizations only show up in some applications and not others?

A: Due to the way that plugins interact with the Mac OS X localization system, a particular localization for a plugin will only be used if the application loading the plugin is also localized into that same language.
