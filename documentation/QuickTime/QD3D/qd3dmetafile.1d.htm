Reference Objects

Labels

ASCII
Reference
Binary
rfrn ( = 0x7266726E )

Data Format

Uns32               refID
refID
A positive integer. The reference object with this refID is linked to a file pointer by means of a table of contents entry that contains both the refID and the file pointer.

Data Size

4

Description

A reference object is used to permit an object defined elsewhere to be referenced at one or more locations in a metafile. See "File Pointers" , for details.

Parent Hierarchy

Shared.

Parent Objects

A reference object sometimes but not always has a parent object.

Child Objects

None.
