Description

This sample code contains the source for the NSPersistentDocument Core Data Tutorial. The tutorial takes you through the steps of building a simple Core Data–based application using NSPersistentDocument and Cocoa bindings. NSPersistentDocument is a subclass of NSDocument that integrates with the Core Data framework. You will find this tutorial useful if you’re using the Core Data framework to create a document-based application.
