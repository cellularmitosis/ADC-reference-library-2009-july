A New Data Handler Routine

In previous releases of QuickTime, when you used the DataHGetMovie routine, the data handler relied on whatever flags it thought appropriate. A new data handler call, DataHGetMovieWithFlags, is introduced in QuickTime 4.1 to allow flags to be passed to a data handler capable of returning movies. These flags correspond to flags normally passed to NewMovie, NewMovieFromFile, and NewMovieFromDataRef, such as newMovieActive.
