Motion
Motion is an application for creating, editing, and rendering sophisticated motion graphics with drag-and-drop ease. With GPU-accelerated 32-bit float rendering, it offers true film-quality output. Its features include automated replication and more than five hundred filters, effects, and particle presets. Developers can use the FxPlug API to create plug-ins for Motion.
