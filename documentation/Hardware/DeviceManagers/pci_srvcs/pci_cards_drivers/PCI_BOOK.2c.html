The gestaltMachineType Value

All NewWorld-based Macintosh computers return the same value (406 decimal) for the gestaltMachineType. Any software that depends on gestaltMachineType to verify that the current Macintosh computer is a valid CPU on which to execute needs to be changed to check for the existence of the required hardware nodes in the device tree. The Macintosh Name Registry provides routines for checking the device tree. The Name Registry is defined in Name Registry
