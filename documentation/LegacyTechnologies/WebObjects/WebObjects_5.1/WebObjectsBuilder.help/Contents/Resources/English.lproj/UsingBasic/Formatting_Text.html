Formatting Text

The top row of the toolbar contains buttons that format the selected text. If no text is selected, they change the setting for text typed.

To change the texts's style to bold (<B>), italics (<I>), underline (<U>) or typewriter (<TT>):

  • Use the commands in the Format > Font menu or the Style buttons.

To change the text's size:

  • Use the Format > Size menu or the Font Size pop-up menu.

To change the text's color:

  • Choose Window > Colors or use the color well.

To choose how to align the text:

  • Use the Format > Alignment menu or the Alignment pop-up menu.

To change the text's font:

  • Chose Format > Font > Show Fonts or use the Font pop-up menu.

In WebObjects Builder, you can apply a font set to a range of text, and the user's browser uses the first font in the set that's available. For example in the set "Monaco, Courier, Fixed", the user's browser will use Monaco if it's available, then Courier, then the browser's default fixed-width font. WebObjects Builder maintains a list of preferred font sets that you can use in any document you edit. To edit your font sets:

  • Choose Format > Font > Show Fonts.

    To add a font set, enter a comma-separated list of font names, and press Return. To delete a font set, select it and press Delete.
