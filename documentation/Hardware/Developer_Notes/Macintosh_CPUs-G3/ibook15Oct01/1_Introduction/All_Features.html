All Features

Here is a list of the features of the iBook computer. Each feature is described in a later chapter, as indicated in the list.

  • Processor speed The clock speed of the microprocessor is 500 or 600 MHz. See "G3 Microprocessor".
  • Bus speed The system bus in the 600-MHz models runs at 100 MHz. The system bus in the 500-MHz model runs at 66 MHz.
  • PowerStep This feature allows the user to slow down the processor speed from 600 or 500 MHz to 400 MHz to conserve power. For more information, see "PowerStep".
  • Cache The microprocessor has a built-in L2 cache consisting of 256 KB of fast static RAM. The clock speed for the L2 cache is the same as the clock speed of the microprocessor. See "Level 2 Cache".
  • Memory The computer comes with 128 MB of SDRAM on the main logic board. One expansion slot accepts a standard SO-DIMM with up to 512 MB of additional memory. See "RAM Expansion".
  • Hard disk storage The computer has a hard disk drive with a storage capacity of 15 or 20 GB. A 30 GB drive is also available. For more information and developer guidelines for alternative hard drives, see "Hard Disk Drive".
  • CD-ROM drive One configuration has a built-in 24x-speed CD-ROM drive. See "CD-ROM Drive".
  • DVD-ROM drive One configuration has a built-in 8x-speed DVD-ROM drive. See "DVD-ROM Drive".
  • DVD-ROM/CD-RW combo drive One configuration has a 4x/4x/6x/24x DVD-ROM/CD-RW combo drive. For more information, see "DVD-ROM/CD-RW Combo Drive".
  • CD-RW drive A built-in 4x/8x/24x CD-RW drive is available as an option. See "CD-RW Drive".
  • Display The computer has a 12.1-inch TFT display with XGA (1024 by 768) resolution. See "Flat Panel Display".
  • Composite video output The computer has an A/V port with a composite video output signal. See "Composite Video Output".
  • VGA video output The computerhas an output port for connecting a VGA video monitor or projector. See "VGA Video Output".
  • Graphics IC The display controller is an ATI RAGE Mobility M128. See "Graphics IC".
  • Video RAM The display controller includes 8 MB of video RAM. See "Graphics IC".
  • Microphone The computer has a built-in microphone. See "Sound System".
  • Battery The computer has one battery bay. The battery uses six lithium ion cells and has a capacity of 3840 mAh at a nominal 10.8 V. Battery life under normal use is up to 5 hours.
  • USB ports The computer has two USB ports. See "USB Ports".
  • FireWire port The computer has an IEEE-1394 FireWire high-speed serial port. See "FireWire Port".
  • Target Disk Mode The computer can act like a FireWire storage device connected to another computer. See "Target Disk Mode".
  • Modem The iBook has a built-in modem that supports 56 Kbps data rate. See "Internal Modem".
  • Ethernet The iBook has a built in Ethernet port for 10Base-T and 100Base-T operation. See "Ethernet Port".
  • Wireless LAN An internal wireless LAN module is available as a build-to-order option or as a user-installable upgrade. See "AirPort Card Wireless LAN Module".
  • Sound The computer has a built-in microphone and stereo speakers; it also provides stereo output signals on the A/V jack. See "Sound System".
  • Keyboard The keyboard has function keys and inverted-T arrow keys. Some of the function keys are used to control the brightness and sound and to eject a disk; the other function keys are user programmable to open applications or files. The keyboard also includes an embedded numeric keypad. See "Keyboard".
  • Trackpad The integrated trackpad includes tap/double tap and drag features. See "Trackpad".
  • Weight With the battery installed, the computer weighs approximately 2.2 kg (4.9 pounds); the exact weight depends on the configuration.
  • Size The iBook computer is 28.50 cm (11.2 inches) wide and 23.03 cm (9.06 inches) deep. Its thickness is 3.40 cm (1.35 inches).
  • Security slot The iBook computer has a security slot for attaching a Kensington locking system or a Noble security cable.
