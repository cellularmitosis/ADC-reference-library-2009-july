Pop-up Menus

A pop-up lets the user choose among several items. The user can choose only one item at time.

To add a pop-up menu to a form:

  • Click the Pop-Up Menu button , and then bind the menu's list, item, and selection attributes.
    • list is the array of objects to choose from.
    • item is a temporary variable. As WebObjects iterates through list to display its items, it holds the item it's currently displaying in the variable that item specifies. You can use that variable in other bindings to stand for the item being displayed.
    • selection is the item the user has selected.

By default, the menu uses item's toString method to generate the names in the menu. To choose a different method or instance variable to generate the names:

  • Bind the menu's displayString attribute to the new value.
