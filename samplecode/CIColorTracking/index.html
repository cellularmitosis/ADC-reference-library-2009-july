CIColorTracking

Version: 1.3

Posted: 2008-04-24

Build Requirements: Xcode version 2.3 Interface Builder version 2.5

Runtime Requirements: Mac OS X v10.5 or later. Macintosh with Intel or G5 processor. Preferably with an nVidia 7300GT or newer graphic card.

View Source Code:

Download Sample (“CIColorTracking.zip”, 2.70M)
Download Sample (“CIColorTracking.dmg”, 3.02M)


Description

The CIColorTracking sample application is an Objective-C applicaton that implements a GPU-based technique to find the location of a uniquely colored object in a scene and then uses the position information to apply video effects.
