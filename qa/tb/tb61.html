Technical Q&A TB61
Localized Versions of Appearance SDK

Q: Where can I find a localized flavor of the system software components in the Appearance 1.0.X SDK? I have scoured the various web sites and my OS SDK disks, and can only find the US flavor.

A: Unfortunately, there are no localized flavors of this software. The only localized flavors of these components ship with Mac OS 8.0 and Mac OS 8.1. (Under Mac OS 8.5 and later, this functionality has been merged into the OS.) These components are not qualified for use on other systems and cannot be licensed for bundling.

[Apr 12 1999]
