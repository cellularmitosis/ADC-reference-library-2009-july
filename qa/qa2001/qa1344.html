Technical Q&A QA1344
DDR SDRAM ICs for PowerBooks and iBooks that use Memory bus slewing

Q: We are designing DDR SDRAM DIMMs for PowerBooks and iBooks that do Memory bus slewing. What DDR SDRAM Component ICs does Apple use in the DIMMs that ship with these products?

A: As of this writing, the following is a list of the DDR SDRAM Component ICs used in the DIMMs that ship with PowerBooks and iBooks that use Memory bus slewing. (This is subject to change at any time.)

Samsung - K4H560838E

Hynix - HY5DU121622AT

Micron - MT46V16M16

[Mar 18, 2004]
