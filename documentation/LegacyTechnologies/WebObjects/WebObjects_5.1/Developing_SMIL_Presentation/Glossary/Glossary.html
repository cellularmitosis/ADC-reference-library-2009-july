Glossary

component, WebObjects
An object (of the WOComponent class) that represents a Web page or a reusable portion of one.
curl
Command-line tool used to transfer files using URL syntax.
QuickTime Player
A multimedia player.
RealPlayer
A multimedia player.
SMIL (Synchronized Multimedia Integration Language)
Language specification that facilitates the integration of various media objects into a single, synchronized presentation.
URI (Uniform Resource Identifier)
The Web naming and addressing technology. A URI is a string of characters that identify a resource. Some typical URI schemes are HTTP and FTP.
URN (Uniform Resource Name)
URNs are URIs that identify a resource regardless of its physical location. URLs are a type of URN.
