Anchors

To create an anchor (<A NAME="...">), a named location within a document that a hyperlink can jump to, do the following:

  1. Choose Elements > Anchor or click the Anchor button , and type the text for the browser to display.

    Tip: You can also select the text for the browser to display and then press the Anchor button.

  2. In the Inspector, set the anchor's name.

To change an anchor to a static hyperlink:

  • In the Inspector, deselect "Named location (anchor) in this document," select "Link to another document," and enter the destination.
