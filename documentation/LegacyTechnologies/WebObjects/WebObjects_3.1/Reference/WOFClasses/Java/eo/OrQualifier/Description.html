Class next.eo.OrQualifier

CLASS DESCRIPTION

Extends:
next.eo.Qualifier

OrQualifier is a subclass of Qualifier that contains multiple qualifiers. When an OrQualifier object is evaluated, it in turn evaluates each of its qualifiers until one of them returns True. If one of its qualifiers returns True, the OrQualifier object returns True immediately. If all of its qualifiers return False, the OrQualifier object returns False.

OrQualifier

public OrQualifier(next.util.ImmutableVector qualifiers)
