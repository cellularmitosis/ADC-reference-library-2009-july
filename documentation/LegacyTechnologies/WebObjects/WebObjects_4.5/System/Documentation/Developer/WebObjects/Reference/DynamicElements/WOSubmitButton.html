WOSubmitButton



Element Description

A WOSubmitButton element generates a submit button in an HTML page. This element is used within HTML forms.


Synopsis

WOSubmitButton { action=submitForm; value=aString; [disabled=aBoolean;] [name=aName;] ... };

Bindings

action
Action method to invoke when the form is submitted.
value
Title of the button.
disabled
If disabled evaluates to true (or YES), the element appears in the page but is not active. That is, clicking the button does not actually submit the form.
name
Name that uniquely identifies this element within the form. You may specify a name or let WebObjects automatically assign one at runtime.
