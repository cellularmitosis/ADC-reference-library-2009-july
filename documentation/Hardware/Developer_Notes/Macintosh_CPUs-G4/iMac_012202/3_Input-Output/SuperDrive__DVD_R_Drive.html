SuperDrive (CD-RW/DVD-R) Drive

One configuration of the iMac (Flat Panel) computer has a SuperDrive: a combination CD-RW and DVD-R drive. The drive uses tray loading of the disc.

The drive can read and write DVD media and CD media, as shown in Table 3-6. The SuperDrive also provides DVD-Video playback with DVD MPEG2 decode.


Table 3-6 Types of media read and written by the SuperDrive (CD-RW/DVD-R) drive


Media type Reading speed Writing speed
DVD-R 2x 2x
DVD-ROM 6x (single layer), 2x (dual layer) –
CD-R 24x 8x
CD-RW 24x 4x
CD or CD-ROM 24x –

Digital audio signals from the SuperDrive can be played through the sound outputs under the control of the Sound Manager.

The SuperDrive uses the ATA4 protocol and is device-select jumpered as Device 1 (slave) in an ATA Device 0/1 configuration.
