Transparent Overlay Media Skins

If you want the media skin to partially cover the source video, you need to fill the area to be transparent with a solid color of a specific RGB value, such as White, Black, %50 Gray, or 100% Blue.

In the following example image ( Figure 18 ), the QuickTime Q has been filled with %50 Gray. The script preferences can be set to adjust the track's operation color and transfer mode for this value, and the center of the Q image will become transparent when the skinned movie is created.
