The Default Case

Hardware acceleration is on by default in Mac OS X version 10.2 for video cards that Apple has determined provide a noticeable benefit without any other drawbacks. In general this applies to video cards with 16MB or more of video RAM. When your Swing applications start up, the Java implementation in Mac OS X determines which video card is installed in your system and compares this against a list of known unsupported video cards. If your card does not appear in, or is commented out of, this list then hardware graphics accelerating is invoked.

If you do nothing at all your Java applications that run on the newer high performance video cards will take advantage of hardware acceleration.
