Level 2 Cache

In the 500 and 600-MHz models, the data storage for the L2 cache consists of 256 KB of fast static RAM that is built into the microprocessor chip along with the cache controller. The built-in L2 cache runs at the same clock speed as the microprocessor.

The 400-MHz model has an external L2 cache, consisting of 512 KB of fast static RAM operating at a clock speed ratio of 5:2.
