ColorSync
ColorSync is an industry-standard color management system that includes services for color matching, color conversion, and gamut checking. Developers of applications or device drivers can use ColorSync to provide fast, accurate, and consistent color calibration, proofing, and reproduction. ColorSync is built into Quartz 2D, and applications using Quartz 2D get the benefit of automatic color management.
