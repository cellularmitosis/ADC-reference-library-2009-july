Technical Q&A GXTY10
Getting Default Settings for a Given Font

Q: I can't figure out how to get the default settings for the features in a given font. Any ideas?

A: An API was added after the release of GX 1.0 to do this: GXGetFontDefaultFeatures(). This API will retrieve those layout features defined as default by a given font. This API call is fully documented in Technote 1028 - Inside Macintosh: GX Series Addenda.

[Jun 01 1996]
