ATA Manager 4.0

Mac OS 8.6 includes ATA Manager 4.0, which has a modular design similar to that of SCSI Manager 4.3. With this new modular design, the ATA Manager and the ATA Interface Modules (AIMs) divide the responsibilities. The ATA Manager is responsible for

  • registering ATA Interface Modules

  • routing each request to the appropriate ATA Interface Modules

  • calling completion routines for asynchronous calls

ATA Interface Modules are responsible for

  • handling each request

  • error handling

  • returning errors
