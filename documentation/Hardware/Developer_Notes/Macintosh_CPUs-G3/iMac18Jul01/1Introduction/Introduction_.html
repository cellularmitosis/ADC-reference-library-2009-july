Introduction

This chapter lists the features of the current models of the iMac computer, with emphasis on the changes from the previous models. It also provides information about a few software issues of interest to developers.


Note: This chapter describes the 500, 600, and 700-MHz retail models of the iMac computer. The 500-MHz educationmodel is similar to the 500-MHz model described in the iMac developer note for September, 2000.
