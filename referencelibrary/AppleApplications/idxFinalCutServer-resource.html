Final Cut Server
Final Cut Server is a client/server application for media asset management and workflow automation. With event-based and data-driven automation, it enables flexible and scalable configuration. Developers can use response actions (including the ability to run any external script or command) and XML to integrate FInal Cut Server with third-party applications.
