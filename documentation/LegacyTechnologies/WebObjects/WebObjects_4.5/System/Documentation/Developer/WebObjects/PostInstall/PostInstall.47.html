Uninstalling on Solaris and HP-UX

On both Solaris and HP-UX, a shell script--named uninstall.sh --is placed in the Library/Executables directory (relative to the directory in which you installed WebObjects). To uninstall WebObjects, you need only log in as root and execute this script. You'll be asked whether you really mean to uninstall WebObjects; type `y' to proceed with the uninstallation, or any other character to abort the process.
