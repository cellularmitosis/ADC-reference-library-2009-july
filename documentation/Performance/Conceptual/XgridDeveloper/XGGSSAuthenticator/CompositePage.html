XGGSSAuthenticator.h


Discussion



This header file describes the interface of the authenticator that handles the Xgrid GSS authentication protocol.

Classes

XGGSSAuthenticator
An XGGSSAuthenticator is used by the XGConnection object to perform GSSAPI authentication. It is a concrete subclass of the abstract XGAuthenticator class.
