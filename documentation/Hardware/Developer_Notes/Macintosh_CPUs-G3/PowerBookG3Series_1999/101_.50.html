User Installation of an Expansion Bay Module

The user can insert a module into the expansion bay while the computer is operating. This section describes the sequence of control events in the computer and gives guidelines for designing an expansion bay module so that such insertion does not cause damage to the module or the computer.

IMPORTANT

The user must not remove a module from the expansion bay while the computer is communicating with the module or, for a module with a disk drive, while the disk is spinning.
