DVD-ROM/CD-RW Drive

A combination DVD-ROM and CD-RW drive is available as an option. The drive has a tray for loading the disc.

The Combo drive can read DVD media and read and write CD media, as shown in Table 3-4. The DVD-ROM/CD-RW drive also provides DVD-Video playback.


Table 3-4 Types of media read and written by the DVD-ROM/CD-RW drive

Media type Reading speed Writing speed
DVD-ROM 8x (CAV) –
CD-R 32x (CAV) 12x (CLV)
CD-RW 24x (CAV) 8x (CLV)
CD-ROM 32x (CAV) –

Digital audio signals from the DVD-ROM/CD-RW drive can be played through the sound outputs under the control of the Sound Manager.

The DVD-ROM/CD-RW drive is an ATAPI drive and is cable-selected as device 0 (master) in an ATA Device 0/1 configuration.
