Main Memory Bus

The main memory bus is a 133-MHz, 64-bit bus connecting the main memory to the Uni-N IC.

Main memory is provided by up to three PC133 DIMMs. Supported DIMM sizes are 16, 32, 64, 128, 256, and 512 MB. The memory slots will accept three 512-MB DIMMs for a maximum memory size of 1.5 GB.

For more information about memory DIMMs, see RAM Expansion.
