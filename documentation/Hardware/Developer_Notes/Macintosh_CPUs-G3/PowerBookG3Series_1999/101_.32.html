Function Keys

The function keys, F1-F12, are used as control buttons for the display and sound; unused function keys are NOPs. These keys are affected by the Function keys checkbox and the Fn key. Table 3-5 is a list of the function keys and their operation as control buttons.

Table 3-5 The function keys as control buttons  

Key name

Control button
F1 Decrease display brightness
F2 Increase display brightness
F3 Decrease speaker volume
F4 Increase speaker volume
F5 Num Lock
F6 Mute speaker
F7 NOP
F8 NOP
F9 NOP
F10 NOP
F11 NOP
F12 NOP
