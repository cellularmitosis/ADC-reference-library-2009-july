Technical Q&A SND09
siOSTypeInputAvailable Format

Q What is the format of the data returned by the siOSTypeInputAvailable selector?

A The format is SoundInfoList. You pass a pointer to a SoundInfoList in the SPBGetDeviceInfo call with the siOSTypeInputAvailable selector, and it returns in that pointer a SoundInfoList structure which contains a short (count) and a Handle (infoHandle) to the OSType list.

[Jan 16 1998]
