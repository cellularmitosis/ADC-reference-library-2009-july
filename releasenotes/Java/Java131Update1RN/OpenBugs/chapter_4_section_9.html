Java Sound

Radar # 2588625

Audio input doesn't work.

Description:

Audio input doesn't work through JavaSound.

Workaround:

Although you cannot currently use JavaSound, Mac OS X does have a very robust audio system. You can find more information about the audio architecture of Mac OS X at http://developer.apple.com/audio. Specific options for providing audio functionality to your Java application in Mac OS X include:

  • Use QuickTime for Java based services. This will work on Mac OS 9, Mac OS X as well as Windows.

  • Use CoreAudioJava based services. This API is for Mac OS X onlyand provides Java classes to access the underlying CoreAudio platform services foraudio and MIDI in Mac OS X. API reference is available at http://qtj.apple.com/pub/audio/docs/index.html.

  • Use JNI to provide a native binding to CoreAudio directly. CoreAudio is the underlying operating system sound API. Information on CoreAudio in general can be found at http://developer.apple.com/audio/pdf/coreaudio.pdf.
