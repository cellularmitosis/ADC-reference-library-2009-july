Technical Q&A USB01
Using Custom creator codes for USB Class Drivers

Q Can I use my own creator code for my USB class driver so that I can associate custom icons with the file?

A For the current releases of Mac OS USB to v1.0.1, you must set the creator code of a USB class driver file, to 'usbd'; otherwise, the driver will not be detected. We are investigating a modification to the driver detection mechanism. When this change occurs, this Q&A will be updated and notice will be posted to subscribers on the USB mailing list. For information on the USB mailing list, go to the Mac OS USB Developers web site.

[Sep 28 1998]
