Tables

To create a table, click the Table button and fill out the New Table dialog that appears.

In the Inspector, you can modify a cell's, row's, or table's attributes, including its size, alignment, or background. You can also add and remove rows and columns in the Inspector.

To select a cell, click in it. To select a table or row, select one of its cells, then click the tag for the table or row in the path view.


Creating a Table

To create a table:

  • Choose Elements > Table or click the Table button , and fill out the New Table dialog that appears.

    The preview pane shows what your table will look like and reflects all the options except the ones for table width and height.
    • To change the number of rows or columns, use the Dimensions fields.
    • To change the table's width or height, choose a unit from the Width or Height menu and enter the size.
    • To add a caption, click the Captioned checkbox and select the caption's position.
    • To change the table's border, spacing (CELLSPACING), or padding (CELLPADDING), enter the size in pixels in the Layout fields.
    • To create headings for the columns (<TD>), select "First row cells are header cells."
    • To create a row that contains a repetition, select "Second row is wrapped in a WORepetition." The repeated row is shaded in blue.

Selecting Items in a Table

To select a cell:

  • Click in the cell.

    To select the next cell, press Tab. To select the previous cell, press Shift-Tab.

To select a contiguous region of cells:

  • Drag across the cells, or click the first cell and Shift-click the last cell.

To select a discontiguous region of cells:

  • Click one cell and Command-click the other cells.

To select a table row:

  • Click in one of the row's cells, then click the <TR> tag in the path view.

To select a table:

  • Click in one of the table's cells, then click the <TABLE> tag in the path view.

Note that selecting all the cells in a row or table is not the same as selecting the row or table. To select a row or table, click its tag in the path view.


Adding and Deleting Rows and Columns

You can add and delete rows and columns with buttons in the Inspectors for tables, rows, and cells. You can also use commands in the Format > Table menu.

To add a row:

  • Select a cell or row, and click either the Add Row Above ( )or the Add Row Below ( ) button in the Inspector.

To add a column:

  • Select a cell, and click either the Add Column Before ( ) or the Add Column After { ) button in the Inspector.

To add multiple rows or columns:

  • Select a cell, and choose Format > Table > Add Rows / Columns.

To delete a row:

  • Select the row to delete (or a cell within that row), and click the Delete Row button ( ) in the Inspector.

To delete a column:

  • Select a cell within he column to delete, and click the Delete Column button ( ) in the Inspector.

To split a cell into two cells:

  • Select the cell, and click either the Split Vertically ( ) or the Split Horizontally ( ) button in the Inspector.

To merge multiple cells into one cell:

  • Select a contiguous region of cells, and click the Merge Cells button ( ) in the Inspector.

    If this button isn't available, the selected cells cannot be merged into one cell.

To wrap an abstract dynamic element (conditional or repetition) around a selected row or cell, click the dynamic element's icon in the toolbar. (See Conditionals and Repetitions.)
