Analog Display Resolutions

Table 3-7 shows the display resolutions, vertical scan rates, and maximum pixel depths supported on analog (CRT) monitors. When power is applied, the monitor is initially set for a display size of 640 by 480 pixels. With a multisync monitor the user can switch the monitor resolution during operation by using the Monitors control panel or the BitDepth and Resolution modules in the control strip.
