Manager Group: Carbon Core


The Carbon Core incorporates essential services that are generally devoid of a user interface. These facilities range from memory management to process management. Of these facilities, Carbon does not support two older Mac OS managers. For the unsupported Vertical Retrace Manager, Apple will provide sample code that demonstrates how to get equivalent functionality using the Time Manager. For the unsupported Patch Manager, Apple is investigating other ways to provide programmatic extensibility.
