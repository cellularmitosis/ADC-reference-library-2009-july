Manager: Shutdown Manager


Carbon does not support the Shutdown Manager. Mac OS X and Mac OS 8 and 9 use separate APIs to handle shutdown, so utility vendors should call the APIs specific to each OS.

Applications continue to receive quit Apple events, but they do not receive notifications of shutdown.
