/*
  Supplying a 'desC' resource lets the driver UI code know what items are
  a checkboxes. It then handles checking and unchecking those checkboxes
  automatically.
  
  The 'desC' resource also gives the ranges of grouped radio buttons so
  that the driver UI code can automatically make sure that one in the 
  group is selected and no others.
  
  If you choose to not supply this resource then you'll have to manage
  radio buttons and checkboxes yourself.
*/
resource kPanelDescriptionType (kFirstPanelDITL, purgeable) {
  {  /* array Checkboxes: 1 element. Item 3 is a check box */
    3
  },
  {  /* array Rads: 1 element. Items 5,6,7 are a group of radio buttons */
    5, 7
  }
};
