Class next.eo.GenericRecord

CLASS DESCRIPTION

Extends:
next.util.NextObject

GenericRecord is the "default" enterprise object class: If an EOModel doesn't have a custom enterprise object class defined for a particular entity, an EODatabaseChannel using that model creates GenericRecords when fetching objects for that entity from the database server. You typically use GenericRecord objects for data that you don't need to add custom behavior to. Because GenericRecord must be usable for any EOEntity in a model, it's the exception to the rule that an enterprise object class may be mapped to no more than one entity. The Enterprise Objects Framework maps GenericRecords to EOEntity objects per instance rather than as a class.
