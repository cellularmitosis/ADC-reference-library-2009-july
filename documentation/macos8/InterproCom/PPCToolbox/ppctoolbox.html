PATH Documentation > Mac OS 8 and 9 > Interapplication Communication > PPC Toolbox

You can use the Program-to-Program Communications (PPC) Toolbox to send and receive low-level message blocks between applications. The PPC Toolbox can be used by multiple applications running on the same computer or across a network.
