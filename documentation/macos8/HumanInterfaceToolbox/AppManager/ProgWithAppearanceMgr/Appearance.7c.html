Programming With the Appearance Manager

Application-Defined Functions

The Appearance Manager supports the following application-defined functions:

  • MyThemeIteratorProc performs a custom response to an iteration over themes installed on a system.
  • MyThemeButtonDrawProc draws a button label.
  • MyThemeEraseProc draws a background.
  • MyThemeTabTitleDrawProc draws a tab title.
  • MyWindowTitleDrawingProc draws a window title.
  • MyMenuItemDrawingProc draws a menu item.
  • MyMenuTitleDrawingProc draws a menu title.
