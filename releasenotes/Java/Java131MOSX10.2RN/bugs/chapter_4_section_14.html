Java Sound

Radar #2588625

Audio input doesn't work.

Description:

Audio input is not implemented through JavaSound.

Workaround:

Although you cannot currently use JavaSound, Mac OS X provides a very robust audio system. You can find more information about the audio architecture of Mac OS X at http://developer.apple.com/audio. Specific options for providing audio functionality to your Java application in Mac OS X include:

  1. Use QuickTime for Java based services. This works on Mac OS 9, Mac OS X as well as Windows.

  2. Use CoreAudioJava based services. This API is for Mac OS X only and provides Java classes to access the underlying CoreAudio platform services for audio and MIDI in Mac OS X.

  3. Use JNI to provide a native binding to CoreAudio directly. CoreAudio is the underlying operating system sound API.
