Description

Demonstrates major features of HITextView, including new features in Tiger. Step by step projects with documentation and code changes show how to build up a basic text editor from scratch using HITextView. Steps include instantiating an HITextView and setting options such as font and spelling panel support, transparency, and scrolling. Also demonstrates use of CFURL based APIs for reading and writing the contents of an HITextView to and from a file.
