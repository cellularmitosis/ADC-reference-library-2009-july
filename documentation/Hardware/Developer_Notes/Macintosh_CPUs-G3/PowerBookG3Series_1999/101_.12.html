Memory Controller and PCI Bridge

The memory controller and PCI bus bridge IC is a Motorola MPC106, also called Grackle. The Grackle IC provides the bus bridge between the processor bus used on the processor module and the PCI bus used for the I/O controllers on the main logic board. The Grackle IC also contains the controller for the main memory.

The main memory bus runs at a clock speed of 66.67 MHz. The internal PCI bus runs at 33.33 MHz. To enhance performance, the Grackle IC supports concurrent transactions on the main memory bus and the PCI bus.
