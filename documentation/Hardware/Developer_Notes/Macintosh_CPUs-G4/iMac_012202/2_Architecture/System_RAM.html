System RAM

The memory subsystem in the iMac supports two slots: one for a 168-pin DIMMs (dual inline memory module) and one for a 144-pin SO-DIMM. The data bus to the RAM and DIMM is 64 bits wide, and the memory interface is synchronized to the MaxBus interface at 100 MHz.


Note: Only the SO-DIMM slot is accessible by the user. See "RAM Expansion"
