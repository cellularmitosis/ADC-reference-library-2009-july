Installing the Software

This chapter provides an overview of WebObjects Deployment installation issues, including what elements of the software need to be installed on a computer, taking the computer's purpose into account. However, it does not give platform-specific instructions. For detailed installation instructions, see the Installation Guide included with your package.

The following topics are addressed:

  • "Choosing What to Install"
  • "Environment Variables in UNIX Platforms"
  • "Confirming That wotaskd Is Active"
  • "Default Adaptor Installation"
  • "Building Adaptors From Source"
