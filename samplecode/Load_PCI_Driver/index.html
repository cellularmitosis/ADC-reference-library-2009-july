Load PCI Driver

Version: 1.0

Posted: 2003-01-14

Build Requirements: N/A

Runtime Requirements: Carbon PowerPC PCI based Macintosh

View Source Code:

Download Sample (“Load_PCI_Driver.zip”, 340.5K)
Download Sample (“Load_PCI_Driver.dmg”, 445.4K)


Description

Load PCI Driver is a utility for PCI driver writers. It lets you to load and unload a Native PCI driver ('ndrv') without having to restart. This is very useful when debugging your native driver. Requirements: PowerPC PCI based Macintosh Keywords: PCI ndrv driver loader, Load PCI Driver
