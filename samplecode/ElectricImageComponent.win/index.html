Description

This sample demonstrates how to build five QuickTime Components; a Graphics Importer, Graphics Exporter, Movie Importer, Movie Exporter and Image Decompressor. These components all work together to allow QuickTime to use Electric Image format image files. Projects for Mac OS X, Windows and Traditional Mac OS are included. Mac OS X: CodeWarrior 8.3 and Project Builder (October 2002) (QT 6.0.2) Windows: Visual Studio.net VC++ 7 and VC++ 6 (QT 6 SDK) Mac OS 9: CodeWarrior 8.3 (Universal Interfaces 3.4.2) Sample Electric Image files are included. Requirements: CodeWarrior 8.3+, Project Builder (December 2002)+, Visual Studio.net 7.0 or Visual C++ 6.0, QuickTime 6 Windows SDK Keywords: Graphics Import, Graphics Export, Movie Import, Movie Export, Decompressor, Codec, QuickTime, Electric Image
