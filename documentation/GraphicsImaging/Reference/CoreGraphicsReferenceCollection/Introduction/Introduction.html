This collection of documents provides the API reference for the Core Graphics technologies, which are fundamental to the Mac OS X graphics and windowing environment. Core Graphics technologies include Quartz 2D, the drawing API that implements a superset of the PDF 1.4 specification; Quartz Display Services, the API for configuring and controlling display hardware; and Quartz Events Services, the API for observing and alterating low-level user input events.
