System RAM

The memory subsystem in the PowerBook computer supports two slots for 144-pin SO-DIMMs (small-outline dual inline memory modules). The data bus to the RAM and DIMM is 64 bits wide, and the memory interface is synchronized to the 60x bus interface at 100 MHz. See RAM Expansion Slots.
