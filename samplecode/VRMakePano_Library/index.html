Description

Version: 1.15 Date: 10/18/2001 This sample contains functions to create QuickTime VR panoramic movies from picture files (i.e. 'PICT', 'JPEG', etc.), GWorlds, or linear QuickTime movies. Hot spot and fast start picture files, GWorlds or movies can be supplied as well. Version 2.0 QuickTime VR cylindrical panoramic movies can be created (in either classic rotated or modern non-rotated orientations), as well a cubic panoramic movies. This code runs on Macintosh OS 8-9, OS X, and Windows.
