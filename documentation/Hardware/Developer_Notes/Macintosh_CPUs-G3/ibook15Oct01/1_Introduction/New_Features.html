New Features

The features that have changed are listed here along with references to the sections that describe them.

  • Processor speed Two configurations have a microprocessor clock speed of 600 MHz. See "G3 Microprocessor".
  • Bus speed The system bus in the 600-MHz models runs at 100 MHz.
  • Memory All configurations come with 128 MB of SDRAM on the main logic board. See "RAM Expansion".
  • Hard disk storage The computer has a hard disk drive with a storage capacity of 15 or 20 GB. A 30 GB drive is also available. For more information and developer guidelines for alternative hard drives, see "Hard Disk Drive".
