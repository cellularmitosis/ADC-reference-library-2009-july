New Features

Here is a list of the features that are new to the Power Mac G4 computer.

  • Microprocessor clock speed: The microprocessor clock frequency is either 733, 800, or 867 MHz. For more information, see "PowerPC G4 Microprocessor".
  • Memory caches: The G4 microprocessor used in the Power Mac G4 computer has an internal 256 KB level 2 cache. The 800 and 867-MHz models also have an external 2 MB level 3 cache. For more information, see "Cache Memory".
  • Graphics card: The graphics card in the AGP-4x slot uses an NVIDIA GeForce2 MX graphics controller. A graphics card that supports two monitors is also available. For more information, see "Graphics Card".
