General Changes

In addition to many bug fixes and some overall improvements, Mac OS X version 10.2 includes some specific enhancements to the Java 2 Platform, Standard Edition version 1.3.1 development environment that you should be aware of. This document outlines changes from previous versions of Java on Mac OS X. It is important to note that the information in this document supersedes any information provided in previous versions of the Release Notes. Release Notes for Java in Mac OS X are available at http://developer.apple.com/techpubs/macosx/Java/java.html.

Overall changes include:

  • Java Web Start and Applet Launcher have been moved from /Applications/Utilities to /Applications/Utilities/Java.

  • Java Plugin Settings is a new application included in Mac OS X version 10.2. It is installed into /Applications/Utilities/Java

  • Application startup time has been improved with the use of the two level namespace by the Java Virtual Machine (VM).

  • Extensive changes have been made to the Java applet support in Mac OS X version 10.2. More information on this is available in “Applet Support Enhancements”.

  • With this update, the mrj.version system property is set to 3.3.
