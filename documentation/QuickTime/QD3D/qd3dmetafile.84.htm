Generic Renderers

Labels

ASCII
GenericRenderer
Binary
gnrr ( = 0x676E7272 )

Data Format

No data.

Data Size

0

Description

A generic renderer performs no rendering functions, but may be used to pick or to accumulate state.

Parent Hierarchy

Shared, renderer.

Parent Objects

View hints (sometimes).

Child Objects

None.
