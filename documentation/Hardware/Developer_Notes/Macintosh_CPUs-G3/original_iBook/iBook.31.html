The RAM Expansion Module

The RAM expansion slot accommodates a standard SO-DIMM (small outline, dual inline memory module) that uses SDRAM devices.

IMPORTANT

A RAM expansion SO-DIMM for the iBook must use SDRAM devices. If the user installs an SO-DIMM that uses EDO or SGRAM devices, the computer will beep several times when the user attempts to restart the computer.

An SO-DIMM for the iBook can contain either 32, 64, or 128 MB of memory. Total RAM capacity using devices currently available is 160 MB and is limited by the space available for the SO-DIMM. The slot can accommodate a 1.25-inch SO-DIMM.
