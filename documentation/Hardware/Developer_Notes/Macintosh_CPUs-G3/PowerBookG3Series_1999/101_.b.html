System Software

The PowerBook computer has newly designed system software that provides Open Firmware booting and Mac OS ROM in RAM. The system software is described in Chapter 5.

The system software that comes with the PowerBook computer is Mac OS 8.6 with the addition of the extensions and control panels required for product-specific features. For a description of the general Mac OS 8.6 release, developers should refer to the Technote for Mac OS 8.6. The technote is available on the Technote web site at

http://devworld.apple.com/dev/technotes.shtml
