The RAM Expansion Module

The RAM expansion slot accommodates a standard SO-DIMM (small outline, dual inline memory module) that uses SDRAM devices.

Important

A RAM expansion SO-DIMM for the iBook computer must use SDRAM devices. If the user installs an SO-DIMM that uses EDO or SGRAM devices, the computer will beep twice when the user attempts to restart the computer.

An SO-DIMM for the iBook computer can contain either 32, 64, 128, 256, or 512 MB of memory. Maximum total RAM capacity using devices currently available is 640 MB.
