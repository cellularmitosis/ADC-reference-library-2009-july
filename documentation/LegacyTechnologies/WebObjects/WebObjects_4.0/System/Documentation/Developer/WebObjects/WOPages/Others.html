Help from Others

OmniGroup's WebObjects Mailing List
The Omni Development group graciously maintains a very useful WebObjects mailing list, on which you can ask questions, answer questions, and share code with other WebObjects developers. This is probably the most useful external source of WebObjects information you'll find.

To subscribe to the WebObjects mailing list, send mail to listproc@omnigroup.com saying (in the body of the message): 

subscribe WebObjects Your Name


WebObjects Mailing List Archive
A complete archive of messages to the mailing list. Start here if you really need the answer now.


WebObjects FAQ
Answers to the most frequently asked WebObjects questions, contributed to by members of the WebObjects mailing list and maintained by Scott Anguish on the Stepwise site.
