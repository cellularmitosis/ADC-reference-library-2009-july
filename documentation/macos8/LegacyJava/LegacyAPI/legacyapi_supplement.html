PATHDocumentation > Legacy API
 
This section applies to Java documentation for Mac OS versions prior to Mac OS X.

JDirect
JDirect is a technology that allows Java code to call native Mac OS code.
JBindery
JBindery is an application that packages Java applications to run on Mac OS computers. You can launch Java applications packaged with JBindery just like a Mac OS application.
