Changes to Graphics Importers

The Photoshop graphics importer now implements the GetDefaultMatrix call, so that applications can locate the position of individual layers in multi-layer files.

The TIFF graphics importer now implements the GetDefaultMatrix call to help applications cope with TIFF fax images that have different horizontal and vertical resolutions.
