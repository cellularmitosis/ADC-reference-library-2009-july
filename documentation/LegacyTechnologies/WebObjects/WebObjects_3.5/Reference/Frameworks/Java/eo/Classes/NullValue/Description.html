Class Description

The NullValue class defines a unique object used to represent null values in collection objects (which don't allow null values). For example, dictionaries fetched by a next.eo.AdaptorChannel contain the NullValue instance for such values. NullValue is automatically translated to null in enterprise objects, however, so most applications should rarely need to account for this class.

NullValue has exactly one instance, returned by the nullValue class method. You can safely cache this instance and use the == operator to test for the presence of a null value: 

NullValue myNull = NullValue.nullValue();
/* ... */
if (value == myNull) {
    /* ... */
}
