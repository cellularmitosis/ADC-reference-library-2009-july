RAM Expansion Slots

The iMac has two RAM expansion slots. One of the slots accepts a standard PC133 168-pin DIMM (dual inline memory modules) and is occupied by the factory-installed RAM. The other slot accepts a standard PC133 144-pin SO-DIMM and is normally used for added RAM.


Note: Only the SO-DIMM slot is accessible by the user. (The access panel is on the bottom of the computer.)

Important

The user should be reminded to observe the usual precautions to avoid damage to the electronic components due to static electricity.

WARNING

The computer must be turned off before RAM modules are removed or inserted. To remind the user, a red LED is visible in the access door. If the red LED is on, power is on, and must be turned off before changing RAM modules.
