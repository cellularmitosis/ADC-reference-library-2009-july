Explanation of Terms

This article describes the availability and support level fields of a listing and explains their meanings and possible values. You can use this information to produce web content that is compatible across multiple platforms or optimized for viewing on iPhone.

Contents:

Availability
Support Level


Availability

Availability tells the versions of Safari in which the element first appeared or was deprecated. Deprecated tags are not guaranteed to be supported in the future. Note that elements available in Safari for Mac OS X and Windows are not automatically available in iPhone OS, and vice versa.
