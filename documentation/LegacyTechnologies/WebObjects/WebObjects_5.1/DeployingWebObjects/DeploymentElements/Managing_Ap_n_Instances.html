Managing Application Instances

This chapter provides a detailed description of Monitor and wotaskd, two tools you use to manage the application instances running on your site, and it explains how instances communicate with wotaskd:

  • "Configuration Files" provides an overview of the configuration files that can be used in your site.
  • "Lifebeats" introduces lifebeats, the mechanism used by wotaskd processes to determine the status of the application instances that they manage.
  • "wotaskd Processes" explains how wotaskd processes ensure that the application instances they manage are always running. It also describes how to restrict access to wotaskd using Monitor.
  • "Starting WebObjects Services" shows you how to configure a computer to start wotaskd and Monitor during its startup process.
