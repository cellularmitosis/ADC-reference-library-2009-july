Accessibility Inspector Release Notes

  • This is one of two new tools for working on accessibility, the other being Accessibility Verifier. Both tools are located in the folder /Developer/Applications/Utilities/Accessibility Tools/

  • The hotkey F5 used by this tool is also used by the new global search feature in the menubar. To prevent this from interfering with the use of Accessibility Inspector, you can temporarily remove the global search by command-dragging it out of the menubar.

  • An older version of this tool was released as sample code under the name "UIElementInspector", and is available at on the Apple Developer website.
