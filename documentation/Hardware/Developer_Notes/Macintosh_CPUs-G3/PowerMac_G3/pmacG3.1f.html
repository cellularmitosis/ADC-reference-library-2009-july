Architecture

This chapter describes the architecture of the Power Macintosh G3 computers, which consists of a main logic board and one of two input/output cards installed in the PERCH connector. It describes the major components of the main logic board: the microprocessor, the custom ICs, and the display RAM. Input and output connectors are discussed in I/O Features
