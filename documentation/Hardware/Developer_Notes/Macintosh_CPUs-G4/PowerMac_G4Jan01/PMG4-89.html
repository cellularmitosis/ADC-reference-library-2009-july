Multiprocessing Services

Some configurations of the Power Mac G4 computer have dual microprocessors. The Multiprocessing Services API allows your application to create tasks that run independently on one or more processors. The Multiprocessing 2.1 SDK includes interfaces and libraries, documentation, demonstration applications, and sample code. You can download the SDK from Apple's developer site on the World Wide Web at

ftp://ftp.apple.com/developer/Development_Kits/Multiprocessing_2.0_SDK.sit.hqx
