Leopard Guides: Storage
System Administration
Mac OS X Server administrators use graphical applications to monitor drives attached to Xserve hardware, create and manage Xserve RAID arrays, administer Xsan file systems. Administrators can also manage Xsan file systems from the command line. Users configured to use Xsan storage can use a graphical application to check their personal space quotas.
