SoundMediaGetSource

This is another version of the Sound media handler that uses the extended form of scheduledSoundCmd to send the source buffer size (in bytes) to the decompressor. This implements the preferred approach. For more information, refer to the Sound Manager 3.3 Release Notes or the document Mac OS Sound available at <http://developer.apple.com/documentation/quicktime/qtdevdocs/RM/pdfframe.htm>
