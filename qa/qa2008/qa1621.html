Technical Q&A QA1621
Opening a Web page using an HREF track in QuickTime Player

Q: Can I open a Web page using an HREF track in QuickTime Player?

A: HREF tracks only work when a movie is played in the QuickTime browser plug-in.

Currently, when a local movie is played in QuickTime Player, it treats an HREF track as an ordinary text track.
