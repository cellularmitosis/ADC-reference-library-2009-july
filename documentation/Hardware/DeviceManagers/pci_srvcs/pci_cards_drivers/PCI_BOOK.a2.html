Hardware Interrupt Mechanisms

To install an interrupt handler, you use InstallInterruptFunctions, which replaces the earlier Slot Manager routine SIntInstall.

The interrupt set ID and interrupt member number values are available as driver-ist properties associated with each device entry in the Name Registry. For a complete discussion of native driver interrupt handling, see Interrupt Management.
