Lists

To create a new unordered or bulleted list (<UL>):

  • Choose Elements > List (unordered) or click the Unordered List button .

To create a new ordered or numbered list (<OL>):

  • Choose Elements > List (ordered) or click the Ordered List button .

Tip: If there's a selection, each line in the selection becomes a list item (<LI>).

In the Inspector, you can set the type of bullet for an unordered list, set the type of numbering and the start value for an ordered list, or change one type of list to another.

When typing in a list, keep the following in mind:

  • To create a new list item, press Return.
  • To create a line break but no new list item, press Shift-Return.
  • To create a new list nested within the original list, press Tab or choose Format > Indent.
  • To move the nesting back one level, press Shift-Tab or choose Format > Outdent.
