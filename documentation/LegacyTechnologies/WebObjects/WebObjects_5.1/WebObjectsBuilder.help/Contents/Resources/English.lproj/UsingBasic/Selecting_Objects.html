Selecting Objects

To select an object, click it or drag across it.

To select a line, triple-click it.

Some HTML elements, such as table rows and cells, cannot be selected directly in the Component view. To select an element like this, click inside it and then click its tag in the path view.

When you're editing raw HTML in the source view, double click a tag to select the tag, and triple-click a tag to select the tag and its matching tag.
