USB Ports and Devices

The USB ports take the place of the ADB and serial I/O ports found on earlier Macintosh computers. Software shims have been added to allow existing applications designed to work with ADB mice and keyboards to work with the equivalent USB devices. See USB Ports .

The AppleVision display uses an ADB connection for computer calibration of the display. The PowerBook G3 computer has no ADB port, and the USB-to-ADB adapter does not work in this capacity, so the user cannot use system software to calibrate the display. The user can still adjust the display manually.
