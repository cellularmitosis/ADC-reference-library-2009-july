Compatibility Issues

While the PowerBook computer has many new features, there should be no compatibility problems with applications and peripherals that operate correctly with earlier PowerBook G3 Series models, with the exceptions described in this section.

SCSI Port Not Present

The PowerBook computer does not have a SCSI port. Any software designed to communicate directly with a SCSI device will not run properly. It is possible that USB converters will be able to support some of those devices. Software that communicates with the SCSI manager will be supported; see SCSI Legacy.
