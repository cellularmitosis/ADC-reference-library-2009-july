Description

The QTKitPlayer application allows you to open and play QuickTime movies, and also to import them. This added capability means that your application will be able to import and display a wide range of media types that QuickTime understands and supports in Mac OS X, including but not restricted to MP4 clips, JPEG images, and audio files.

This QTKitPlayer application also includes editing capabilities more extensive than just cut, copy, and paste: features such as add, add scaled, replace, and trim, which will enable you to edit QuickTime movies with greater control and precision.

Beyond importing and enhanced editing capabilities, the QTKitPlayer application also implements additional controls for movie playback, such as Start, Stop, Go to Beginning and Go to End, Step Forward, Step Backward, Set Fill Color, Show Controller, Loop, Loop Palindrome, and Clone. All of these are available in the QTKit framework.
