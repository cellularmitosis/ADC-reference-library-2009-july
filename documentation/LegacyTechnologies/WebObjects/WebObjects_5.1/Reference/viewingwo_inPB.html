Viewing WebObjects API Reference in Project Builder

To view the WebObjects API Reference documentation in Project Builder, first download the TAR archive (available under the Reference category on http://developer.apple.com/techpubs/webobjects/webobjects.html). Unarchive this file and copy the contents of the resultant Javadoc folder to /Developer/Documentation/WebObjects/Reference. When asked if you want to replace the index.html file currently in the destination directory, choose Replace. Restart Project Builder and select WebObjects Help from the Help menu. This will load the WebObjects API Reference in Project Builder.
