RAM Expansion Slots

The computer has two RAM expansion slots that accommodate standard SO (small outline) DIMMs using SDRAM devices. One of the slots is normally occupied by a factory-installed SO-DIMM. The other slot is available for a user-installed SO-DIMM.

Note

Some configurations come with factory-installed SO-DIMMs in both slots.

RAM expansion SO-DIMMs for the PowerBook G4 computer must be PC100 compliant and must use SDRAM devices. If the user installs an SO-DIMM that uses EDO devices, the boot process will fail when the user attempts to restart the computer and the computer will not operate.

An SO-DIMM using currently-available parts can contain either 64, 128, or 256 MB of memory. The computer can support up to 512 MB total RAM using the highest-density devices available.
