Ordered Lists

Click to create a new ordered list (<OL>). If there is a selection, each line in the selection becomes a list item (<LI>). You can use the Inspector to change the list to an unordered list (<UL>). You can also change the way in which lists appear; for example, displaying an ordered list in Roman numerals.

Typing in an ordered list is the same as typing in an unordered list (see Unordered Lists above.)
