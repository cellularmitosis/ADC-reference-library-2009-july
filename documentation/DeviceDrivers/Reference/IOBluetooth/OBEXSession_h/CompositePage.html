IOBluetoothOBEXSession

Overview

This header contains the definition of the OBEXSession class. An instance of the OBEXSession class represents an OBEX session, independent of the underlying transport. The class provides methods to manipulate a raw OBEX session. If you're working with Bluetooth devices, you are more interested in the IOBluetoothOBEXSession subclass of this class.

For more information about the OBEXSession class, see Working With Bluetooth Devices.


Classes

OBEXSession

Object representing an OBEX connection to a remote target.
