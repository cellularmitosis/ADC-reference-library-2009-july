The Direct to Web Factory

Direct to Web pages are created by the Direct to Web factory, an instance of the D2W class (see Figure 3-3). This object creates pages by instantiating a Direct to Web context and a Direct to Web template for each page. "Rendering a Direct to Web Page: An Example", shows how the factory generates a query page.
