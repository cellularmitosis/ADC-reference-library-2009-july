Read Me - SOAPServer

SOAP Client (Simple Object Access Protocol)

Example code from WWDC 2001, session 607 - Practical Uses of XML with WebObjects.
This demo is part of a pair and is meant to be used with the WebObjects example SOAPServer.

An example of a school application placing a purchase order with a vendor via XMLSOAP.
Also, a nice example of web design using WebObjects.

Project Requirements

Mac OS X
WebObjects 5 Developer Tools
