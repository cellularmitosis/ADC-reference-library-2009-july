Description

This short sample code implements a simple custom push button using the modern HIView APIs. It can be used as a skeleton for more ambitious custom controls

This sample code has been modernized and updated to include a project that produces a universal binary. No code changes were required for it to run correctly on the Developer Transition Systems.
