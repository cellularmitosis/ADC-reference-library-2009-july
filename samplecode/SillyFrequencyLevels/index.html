Description

SillyFrequencyLevels is a simple demonstration of the new QuickTime Audio APIs to obtain real-time measurements of the average audio output power level in one of more frequency bands.

This sample specifically uses the SetMovieAudioFrequencyMeteringNumBands and GetMovieAudioFrequencyLevels APIs.

This project builds a Universal Binary application.
