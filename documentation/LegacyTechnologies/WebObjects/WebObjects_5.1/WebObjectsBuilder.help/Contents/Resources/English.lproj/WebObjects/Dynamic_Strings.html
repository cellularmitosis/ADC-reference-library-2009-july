Dynamic Strings

A dynamic string (WOString) prints the value of a string variable or a method that returns a string. It can be in any other HTML element that can contain text.

To create a dynamic string:

  • Choose WebObjects > WOString or click the WOString button .

To bind a dynamic string to its value without using the attribute menu:

  • Drag from the key in the object browser to the dynamic string's center binding box.
