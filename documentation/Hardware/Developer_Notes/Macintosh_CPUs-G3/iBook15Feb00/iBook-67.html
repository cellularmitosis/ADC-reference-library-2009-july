Other New Drivers

The system software includes new drivers for the Ethernet port and the AirPort Card wireless LAN module.

The new Ethernet driver supports 100Base-T operation. The Ethernet driver uses the Open Transport Data Link Provider Interface (DLPI). The Ethernet driver also supports Wake on LAN.

The driver for the AirPort Card wireless LAN module is not part of the Mac OS ROM image but resides in the Extensions folder.
