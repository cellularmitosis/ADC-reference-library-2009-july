ReadMe for TremoloUnit
-----------------------
version 1.0 (November 2006)

TremoloUnit is a C++ sample project based on Core Audio SDK version 1.4.3.
It demonstrates how to build a simple effect audio unit with a generic view. The
TremoloUnit project corresponds to a tutorial in Audio Unit Programming Guide,
available in the ADC Reference Library at this location:

http://developer.apple.com/documentation/MusicAudio/Conceptual/AudioUnitProgrammingGuide/


Sample Requirements
-------------------
This sample project requires:
	Mac OS X v10.4 or later
	Core Audio SDK v1.4.3 or later
	Xcode 2.4 or later
	
The current version of the Core Audio SDK is installed when you perform a full 
installation of the current version of Xcode Tools. You can also obtain the SDK 
from this location, using your developer connection password:

http://connect.apple.com/

On the Developer Connection website, click Downloads and then Audio. Be sure to 
download the most recent version of the SDK.
