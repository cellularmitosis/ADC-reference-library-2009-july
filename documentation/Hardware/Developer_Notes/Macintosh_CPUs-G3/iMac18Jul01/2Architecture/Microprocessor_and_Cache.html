Microprocessor and Cache

The microprocessor is a PowerPC G3 with a built-in level 2 (L2) cache.


G3 Microprocessor

The G3 processor used in the iMac is a PowerPC 750CX. The 750CX microprocessor has several features that contribute to superior performance, including:

  • on-chip first level (L1) caches, 32 KB each for instruction cache and data cache
  • an on-chip second level (L2) cache consisting of 256 KB with a clock speed ratio of 1:1
  • a microprocessor core optimized for Mac OS applications

Level 2 Cache

The data storage for the L2 cache consists of 256 KB of fast static RAM that is built into the microprocessor chip along with the cache controller. The built-in L2 cache runs at the same clock speed as the microprocessor.
