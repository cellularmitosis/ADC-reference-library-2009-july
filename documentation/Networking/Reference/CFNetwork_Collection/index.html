CFNetwork Reference Collection

CFNetwork is a framework in the Core Services framework that provides a library of abstractions for network protocols. These abstractions can be used to perform a variety of network tasks, such as working with BSD sockets, creating encrypted connections using SSL or TLS, resolving DNS hosts, working with HTTP and FTP servers, and managing Bonjour services.
