Description

Developers looking for the latest version of the Audio Unit Effect Templates should download the Core Audio SDK 1.4.2+.

The Core Audio SDK can be downloaded by logging into your ADC Developer Connection account, selecting Downloads then Audio.

Alternatively, it can be found at the following URL: http://developer.apple.com/audio/download/

Contents: Xcode templates for creating Audio Unit Effects. Three separate Xcode templates are provided to create Audio Units with a Generic, custom Cocoa, or custom Carbon view.

Note: Requires Core Audio SDK 1.4.2 or greater.
