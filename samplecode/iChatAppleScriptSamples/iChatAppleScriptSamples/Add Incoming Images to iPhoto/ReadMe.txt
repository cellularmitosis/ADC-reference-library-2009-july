Add incoming image to iPhoto.applescript
v1.0

This script demonstrates an AppleScript "Incoming File Transfer" event handler for iChat. When this script is installed, it will automatically import JPGs, GIFs, and PNGs into your iPhoto library.
	
To install this script:

1. Copy the script "Add incoming image to iPhoto.applescript" into ~/Library/Scripts/iChat

2. In iChat Preferences, go to Alerts

3. Choose "Incoming File Transfer" for "Event"

4. Check "Run AppleScript" checkbox

5. Select "Add incoming image to iPhoto.applescript" from the "Run AppleScript" dropdown
	
The script will run every time a non-inline file image is received.
