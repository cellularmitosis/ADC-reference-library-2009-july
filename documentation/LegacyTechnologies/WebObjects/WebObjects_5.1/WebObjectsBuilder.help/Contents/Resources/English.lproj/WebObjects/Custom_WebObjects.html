Custom WebObjects

You can include a custom WebObject that isn't available on the toolbar. To create a custom WebObject:

  • Click in the toolbar and enter the class name.

    If WebObjects Builder recognizes the class, it displays its attributes in the inspector. Otherwise, use the inspector to add them (see Binding Elements).

Tip: The WOExtensions palette (see Using Palettes) contains several pre-defined custom WebObjects elements you can use in a component.
