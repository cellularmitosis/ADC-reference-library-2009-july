MEMMOVE(3)               BSD Library Functions Manual               MEMMOVE(3)

NAME
     memmove -- copy byte string

LIBRARY
     Standard C Library (libc, -lc)

SYNOPSIS
     #include <string.h>

     void *
     memmove(void *s1, const void *s2, size_t n);

DESCRIPTION
     The memmove() function copies n bytes from string s2 to string s1.  The two strings may overlap; the
     copy is always done in a non-destructive manner.

RETURN VALUES
     The memmove() function returns the original value of s1.

SEE ALSO
     bcopy(3), memccpy(3), memcpy(3), strcpy(3)

STANDARDS
     The memmove() function conforms to ISO/IEC 9899:1990 (``ISO C90'').
