Support for M3U Playlists and .CEL Files

QuickTime 4.1 can import M3U playlists. This enables you to play the list of MP3 files in sequence.

QuickTime can now recognize and import .CEL files. A .CEL file is virtually identical to a .FLC file. (A .FLC file is an Autodesk animator file.) The only difference between a .CEL file and a .FLC file is the extension, as far as QuickTime is concerned.
