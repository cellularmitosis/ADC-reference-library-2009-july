Cache Memory

In addition to the 256-KB L2 cache built into the PowerPC G4 microprocessor, the processor card also has an external level 3 (L3) backside cache. The L3 cache consists of 2 MB of high-speed SRAM runing at a clock speed of 250 MHz (4:1 ratio).


Note: The Xserve computer does not use jumpers to control the clock speeds of the processor and cache.
