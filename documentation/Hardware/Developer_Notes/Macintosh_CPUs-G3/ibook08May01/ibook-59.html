RGB Monitors Supported

The computer supports RGB monitors that use the DDC (display data channel) standard for identification. Older monitors that do not support the DDC standard are not supported. The following Apple monitors are not supported:

  • Multiple Scan 17

  • Multiple Scan 20

  • AudioVision 14

  • Apple Hi-Res RGB

  • Apple 16" Color

  • Apple Hi-Res Monochrome

  • Macintosh 12" RGB
