Description

This group of utilities is stand alone code to help developers work more easily with the HID Manager for Mac OS X with both MachO and CFM support. It provides a high level abstract of the current HID interface that sacrifices slight flexibility in favor single function set up, device and element retrieval, current value and event query, and configuration. This sample is the source code for the Utilities and does not provide a sample of the utilization. There are other samples which provide complete working examples of a HID device explorer and a simple game implementation. More samples will be forth coming shortly. Mac OS X, Project Builder (CodeWarrior 7 for sample CFM projects) Requirements: We hope this helps people get up and running with HID Manager in a quick and painless manner.
