AirPort Card Wireless LAN Module

The iBook computer supports the AirPort Card, an internal wireless LAN module. The AirPort Card is available as a build-to-order option or as a user-installable upgrade through The Apple Store.

By communicating wirelessly with a base station, the AirPort Card can be used for internet access, email access, and file exchange. A base station provides the connection to the internet or the bridge between the wireless signals and a wired LAN. Software included with the AirPort Card enables a Macintosh computer that has an AirPort Card installed to act as a base station. The user also has the option of purchasing an AirPort Base Station that can be connected to the wired LAN or to a telephone line by way of its built-in 56k hardware modem.

The AirPort Card transmits and receives data at up to 11 Mbps. It is also interoperable with some older wireless LANs, as specified in Hardware Components.
