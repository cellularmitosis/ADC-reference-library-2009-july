Manager: Picture Utilities


Carbon supports the Picture Utilities. However, on Mac OS X, the Picture Utilites are supported only since Mac OS X version 10.1. CalcColorTable, DisposeColorPickMethod, and InitPickMethod callbacks remain ignored in Mac OS X.
