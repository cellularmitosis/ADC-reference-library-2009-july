Introduction

This chapter outlines the features of the PowerBook G4 computer.

The latest configurations are different from the previous ones in three respects.

  • All configurations have a built-in DVD-ROM/CD-RW Combo drive. See "DVD-ROM/CD-RW Combo drive".
  • The AirPort Card is included in all configurations. See "AirPort Hardware".
  • The computer comes with 256 or 512 MB of main memory. See "RAM Expansion Slots".
