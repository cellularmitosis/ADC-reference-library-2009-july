Architecture

This chapter describes the architecture of the newest iMac computer. It includes information about the major components on the main logic board: the microprocessor, the other main ICs, and the buses that connect them to each other and to the I/O interfaces.

Note

This chapter describes the 500 and 600-MHz models of the iMac computer. For a technical description of the 400-MHz model, please refer to the previous iMac developer note.
