Video Services Library

The Macintosh Video Services Library (VSL) provides video interrupt services for vertical blanking, horizontal blanking, and other tasks. It also contains a utility that can be used by graphics drivers that respond to hardware cursor calls as described in Supporting the Hardware Cursor.
