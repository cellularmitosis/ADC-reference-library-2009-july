Cyberdog is a modular architecture for integrating existing and future network protocols and services into a unified experience for the Internet user. Cyberdog is not supported on Carbon.

 
 

   
Cyberdog Programmer’s Kit
The book contains an overview of the Cyberdog architecture, code examples and tutorial steps illustrating how to create Cyberdog components, and a complete reference to the C++ classes and methods that make up the Cyberdog programming interface.

Creating a Cyberdog Service
This document is an addendum to Part 2 of the Cyberdog Programmer's Kit. It outlines the steps you take to implement a Cyberdog service, which provides a network protocol. A network protocol defines the conventions for communication between interconnected computers.
