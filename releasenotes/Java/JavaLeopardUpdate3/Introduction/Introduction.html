Java on Mac OS X v10.5 Update 3

Java for Mac OS X v10.5 Update 3 updates the Java Web Start and Java Applet components on all Intel-based and PowerPC-based Macs for J2SE 5.0 and J2SE 1.4.2. Java SE 6 support is available on 64-bit, Intel-based Macs only. This release is only for Mac OS X v10.5.4 and later, and should not be installed on earlier versions of Mac OS X.

Who Should Read This Document?

Any developer who wants to develop Java applications on Mac OS X v10.5 should read this document as various issues and fixes found in this release may affect your application.
