Class next.eo.AndQualifier

CLASS DESCRIPTION

Extends:
next.eo.Qualifier

AndQualifier is a subclass of Qualifier that contains multiple qualifiers. When an AndQualifier object is evaluated, it in turn evaluates each of its qualifiers until one of them returns False. If one of its qualifiers returns False, the AndQualifier object returns False immediately. If all of its qualifiers return True, the AndQualifier object returns True.

CONSTRUCTORS

AndQualifier

public AndQualifier()
public AndQualifier(next.util.ImmutableVector qualifiers)

Returns a newly-allocated and initialized AndQualifier object. If qualifiers is supplied, this method initializes the receiver with the qualifiers qualifiers.
