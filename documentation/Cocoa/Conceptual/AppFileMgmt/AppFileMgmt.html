Introduction to Application File Management

Contents:

Organization of This Document


This topic describes the Application Kit’s facilities for representing file system objects (files and directories) and allowing users to interact with the file system.

Organization of This Document

This document has the following articles:

  • Concepts

    • “File Wrapper” describes the object that encapsulates a file, directory, or link.

    • “The Save and Open Panels” describes the panels that you run to let users specify files (and sometimes directories) to save and open.

  • Tasks

    • “Working With File Wrappers” describes how to make and manage file wrappers.

    • “Working With Directory Wrappers” describes how to make and manage directory wrappers.

    • “Using a Save Panel” explains how to create and display a save panel.

    • “Using an Open Panel” explains how to create and display an open panel.

    • “Getting the Current Selection” describes how to find out what the currently selected file or directory is.

    • “Filtering Out Browser Items” explains how to make specific items in the browser unselectable.

    • “Configuring a Choose Dialog” describes how to configure an Open panel as a Choose dialog.

    • “Managing Accessory Views” explains how to make, add, and manage accessory views.

  • See Also

    • Document-Based Applications Overview

    • Low-Level File Management Programming Topics
