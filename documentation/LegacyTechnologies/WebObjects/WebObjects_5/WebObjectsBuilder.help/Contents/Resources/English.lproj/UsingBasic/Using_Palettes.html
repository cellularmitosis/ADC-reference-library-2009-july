Using Palettes

A palette is a collection of resources (such as images, static or dynamic HTML elements, and components). You can drag elements from a palette to a component. You can also drag elements from a component to a custom palette to store them.

To open the palette window:

  • Choose Window > Palette or click the Palette on the toolbar.

The Palettes pop-up menu at the bottom of the palette window contains the palette commands. The icons at the top of the Palette window show the available palettes. To select a palette, click its icon. Several pre-configured palettes are provided.

To create a new palette:

  • Choose New Palette from the Palettes pop-up menu.

To add an existing palette to the palette window:

  • Choose Open Palette from the Palettes pop-up menu.

To remove a palette from the palette window:

  • Select the palette in the palette window, and choose Close Palette from the Palettes pop-up menu.

    Note that you cannot close a pre-configured palette.

Editing a Palette

A palette can be either editable or read-only.

  • An editable palette is a palette that you can add items to. When you drag an item from it, the item is removed from the palette. An editable palette has a white background
  • A read-only palette is palette that you cannot add items to. When you drag an item from it, the item remains on it and is copied to your component. A read-only palette has a a gray background.

To make a palette editable or read-only:

  • Choose Make Editable or Make Read-Only from the Palettes pop-up menu.

    Note that you cannot make pre-configured palettes editable.

To add an item from a component to a palette:

  • Option-drag elements from the component window to the palette.

To add a file such as a component, an image, or an EOModel, to a palette:

  • Drag the item from the Finder onto the palette.

To change an item's name:

  • Select its name and type.

To change an item's icon

  • Drag an image file from the Finder onto the item's icon.

To change a palette's icon:

  • Drag an image file from the Finder onto the palette's icon

To save a palette:

  • Choose Save Palette or Save Palette As from the Palettes pop-up menu.
