OTLLCTest

Version: 1.0

Posted: 2003-07-22

Build Requirements: N/A

Runtime Requirements: Carbon See the readme file

View Source Code:

Download Sample (“OTLLCTest.zip”, 56.8K)
Download Sample (“OTLLCTest.dmg”, 120.9K)


Description

Sample program demonstrating how to read and write data packets directly to an ethernet driver using the Open Transport XTI API. The sample program implements both send and receive. Demonstrates the following. 1. How to determine which port AppleTalk is currently using 2. List available ethernet drivers 3. Using Option Management to enable raw mode for both original and mentat template based drivers. Development - Requires CodeWarrior IDE v2.1 or greater along with OT 1.1.2 or greater. For testing with ethernet hardware not using the mentat template, requires a minimum of SSW 7.5.3 and OT 1.1.1. If an ethernet driver that has a driver based on the mentat template, then SSW 8.0 with OT 1.2 is required. Requirements: See the readme file Keywords: Ethernet, raw mode, DLPI, enet, multicast, MDATA, LLC, 802.2, 802.3, OTLLCTest
