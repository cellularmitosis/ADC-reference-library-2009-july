Host Configuration

 This page displays the hosts that Monitor is currently aware of, the status of each host, and whether that host currently can run instances of WebObjects applications. It also displays the number of instances currently running on each host.

To Access: Click the Hosts button in the Monitor banner.

A host machine can run WebObjects applications if it has the WebObjects Deployment packages installed and is running the MonitorProxy service. In order to run an instance on a host remote from Monitor, you must add the host in this section (see "Deploying on Multiple Hosts"). You should use alphanumeric DNS names to refer to hosts. If you enter the name of a host that does not exist, it can take Monitor up to 30 seconds to respond that it cannot contact that host.
