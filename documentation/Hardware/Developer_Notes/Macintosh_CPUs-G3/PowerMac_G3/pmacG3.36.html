AC Line Input

  • Line voltage: 100-120 volts alternating current (V AC) and 200-240 V AC, RMS, single phase, manually set by voltage selector switch
  • Frequency: 50-60 Hz
  • Power: 960 volt-amperes (VA) maximum continuous; 1070 VA peak input

AC Line Output

  • Output receptacle: 100-120 V AC; 3 amperes (A) maximum, or 200-240 V AC; 1.5 amperes (A) maximum (determined by actual input voltage)

DC Power

  • Continuous output: 161 W
  • Peak output (for 12 seconds at startup): 210 W
