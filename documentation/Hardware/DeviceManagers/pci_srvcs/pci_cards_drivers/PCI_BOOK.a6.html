Native Driver Framework

All native (PowerPC) device drivers are Code Fragment Manager (CFM) fragments with the following general features:

  • CFM container format
  • CFM programming interfaces exported from the driver to Mac OS
  • CFM programming interfaces imported by the driver from Mac OS

Generic drivers are CFM fragments that work with the Device Manager and the Driver Loader Library; family drivers are CFM fragments that use other mechanisms. Generic and family drivers are distinguished in Generic and Family Drivers. The general characteristics of both kinds of native drivers are briefly summarized in the next sections.
