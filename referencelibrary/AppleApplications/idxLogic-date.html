Logic
Logic contains a comprehensive set of music creation tools for MIDI timing and synchronization, mixing and automation with total recall, and nondestructive real-time editing of both audio and MIDI. It lets the user mix audio and MIDI data while calculating edits such as quantization, transposition, and note length changes in real time. The user can also resize, solo, mute, loop and mix audio and MIDI objects in real time without stopping the sequencer. Developers can extend the features of Logic by writing Audio Unit plug-ins.
