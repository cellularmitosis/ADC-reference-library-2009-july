Integrating Your Code Into the Request-Response Loop

On each cycle of the request-response loop, an application initiates a series of messages to objects that are participating in request handling. These messages comprise "hooks" allowing you to integrate your code and affect the outcome of the request-handling process. This chapter describes the methods invoked by these messages.
