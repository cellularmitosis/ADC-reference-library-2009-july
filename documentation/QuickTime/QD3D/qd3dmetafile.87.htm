Texture Shaders

Labels

ASCII
TextureShader
Binary
txsu ( = 0x74787375 )

Data Format

No data.

Data Size

0

Description

A texture shader is used to apply a texture to a surface in shading.

Parent Hierarchy

Shared, shape, shader, surface shader.

Parent Objects

None.

Child Objects

Pixmap texture object. A texture shader always has one child object.
