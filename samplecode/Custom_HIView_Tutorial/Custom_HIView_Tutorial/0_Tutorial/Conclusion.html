Tutorial: Writing a custom HIView

F) Conclusion

At this point, our custom HIView is pretty much complete. Depending on the kind of custom HIView you want to write, you may have to implement text input and focus management, as well as accessibility support.

Where to go next? There are numerous custom HIView sample codes available on the ADC website at HIToolbox Sample Code. You may want to study them to see which ones are close to the ones you have in mind.

Furthermore, if you prefer C++ to C, you should look at the HIFramework which is a set of C++ classes which simplify greatly the writing of a custom HIView. HIFramework is available in /Developer/Examples/Carbon/HIFramework/ and it is also available the ADC website at HIFramework Sample Code, please check which one is the most recent before using it.
