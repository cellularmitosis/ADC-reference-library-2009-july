Join

Extends:
next.util.NextObject

Class Description

A Join describes one source-destination attribute pair for a Relationship. See the Relationship class specification for more information and for examples.

Constructors

Join

public next.eo.Join(next.eo.Attribute source, next.eo.Attribute destination)

Creates and returns a new Join with the given source and destination attributes. See the Relationship class specification for an example of creating a relationship using Joins.

See also: addJoin (Relationship)

Instance Methods

destinationAttribute

public next.eo.Attribute destinationAttribute()

Returns the destination ("right") attribute used by the join.

See also: destinationAttributes (Relationship)

isReciprocalToJoin

public boolean isReciprocalToJoin(next.eo.Join otherJoin)

Returns true if this join's source attribute is equal to otherJoin's destination attribute and otherJoin's source attribute is equal to this join's destination attribute. This is known as a back-referenc ing join.

See also: inverseRelationship (Relationship)

sourceAttribute

public next.eo.Attribute sourceAttribute()

Returns the source ("left") attribute used by the join.

See also: sourceAttributes (Relationship)
