Main RAM Expansion

The iMac computer has two RAM expansion slots that accommodate standard SO (small outline) DIMMs using SDRAM devices.

RAM expansion SO-DIMMs for the iMac must use SDRAM devices. If the user installs an SO-DIMM that uses EDO or SGRAM devices, the computer will beep several times when the user attempts to restart the computer.

An SO-DIMM for the iMac can contain either 16, 32, or 64 MB of memory.

Total RAM capacity using devices currently available is 128 MB and is limited by the space available for the SO-DIMMs. The bottom slot can accommodate a 1.5-inch SO-DIMM. The top slot can accommodate a 2.0-inch SO-DIMM.

Future devices with higher density may allow expansion beyond the current 128-MB limit.
