Leopard Reference Library: Audio
Apple Applications
Apple provides APIs, file format specifications, and other resources to provide developers with opportunities to interact with and extend various Apple applications. Core Audio's audio units let developers build signal-processing and device-control plug-ins to enhance Apple's audio applications.
