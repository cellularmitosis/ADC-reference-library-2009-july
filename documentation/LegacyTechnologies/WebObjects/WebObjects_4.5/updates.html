WebObjects Programming Topics
These all-new documents are short, focused discussions on a wide variety of subjects of interest to WebObjects and Enterprise Objects Framework programmers. New topics will be written and added to this set on a frequent, regular basis.

What's New in WebObjects 4.0
This document, which describes the changes made in WebObjects since the previous version of the product, is all new for this release.

What's New in Enterprise Objects Framework 3.0
This document, which describes the changes made in Enterprise Objects Framework since the previous version, is all new for this release.

Getting Started With WebObjects
The tutorials in this book have been rewritten to reflect the latest WebObjects tools and APIs.

WebObjects Developer's Guide
This book contains an all-new introductory chapter that describes WebObjects, discusses how WebObjects applications work (including Java Client applications), and presents an overview of the major parts of a WebObjects application. In addition, the WebObjects Developer's Guide has been thoroughly updated to reflect the current APIs.

Enterprise Objects Framework Developer's Guide
Since the previous version of EOF, this book has been completely rewritten to reflect Java Client and the new APIs. As well, the book now presents code excerpts and method names in both Java and Objective-C.
