Confirm That the WebObjects Processes Are Running

On Mac OS X, open /Applications/Utilities/Terminal. At the command line, run ps -aux. Verify that javawoservice.sh is running.

On Windows 2000, open the Services control panel and verify that Apple WebObjects Monitor 5 and Apple WebObjects Task Daemon 5 running. In addition confirm that their startup mode should is set to Automatic.

On Solaris type ps -eaf in a shell window and verify that javawoservice.sh is running.
