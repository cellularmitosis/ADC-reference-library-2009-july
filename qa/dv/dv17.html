WS PDS Card and SCSI Manager 4.3

Q: Is there a version of SCSI Manager 4.3 that is compatible with the Apple Workgroup server PDS card?

A: SCSI Manager 4.3 is built into the ROMs on some machines (the Quadra 840 AV, the Quadra 660 AV, and the Power Macs) and is available as part of System 7.5 for other Quadras. System 7.5.1 (System 7.5 with the update) supports some additional machines, including the Apple Workgroup 915, but the PDS card in the WS95 is only supported under A/UX, and A/UX doesn't support SCSI Manager 4.3.

[Jul 01 1995]
