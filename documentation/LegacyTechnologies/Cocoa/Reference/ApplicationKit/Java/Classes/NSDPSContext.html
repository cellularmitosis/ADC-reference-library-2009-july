NSDPSContext

Inherits from
NSGraphicsContext : NSObject
Package
com.apple.cocoa.application
Companion guide
Basic Drawing

Overview

NSDPSContext provides the context for the Display PostScript graphics environment. Display PostScript is no longer supported. Use NSGraphicsContext instead.
