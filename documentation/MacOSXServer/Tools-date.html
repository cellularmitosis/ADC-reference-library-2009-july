Leopard Guides: Mac OS X Server
Tools
Mac OS X Server includes Xcode and other free tools for developers to prototype, compile, debug, and optimize software. Xcode, Apple's integrated development environment, can be used with Cocoa, Carbon, or Java. Server administrators use other tools to manage users and groups and file sharing, set up and monitor services, and administer Xserve, Xserve RAID, and Xsan file systems.
