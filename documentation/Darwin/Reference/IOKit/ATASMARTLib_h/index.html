Overview

This header includes functions and data types related to ATA SMART (Self-Monitoring Analysis Reporting Technology) support for detecting problems with ATA hard drive prior to failure.

The functions in the IOATASMARTInterface are divided into two categories:

  • Mandatory API support.
    If the device claims SMART feature set compliance, it must implement these functions.
  • Optional API support.
    If the device claims SMART feature set compliance, it may implement one or more of these functions. Please consult the technical manual for the device to see what functions are supported.
