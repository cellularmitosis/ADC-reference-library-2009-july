Conditionals

A conditional (WOConditional) displays its contents only if a particular condition is true. It has two main attributes

  • The condition attribute controls whether the contents are displayed. By default, the contents are displayed if condition is true.
  • The negate attribute lets you reverse the effect of condition and display the contents if the condition is false instead of true. It's useful when you want to display one chunk of text when condition is true and other chunk of text when condition is false.

    If negate is false, the contents are displayed if condition is true. If negate is true, the contents are displayed if condition is false.

To create a condition that does not include a table row:

  1. Click the WOConditional button and add items to the repetition.
  2. Drag from a Boolean key to the white box beside the conditional's icon

    This binds the condition attribute to the key.
  3. If you want the contents displayed only if condition is false, click the plus sign to change it to a minus sign .

    This sets the negate attribute to true.

When you create a condition around a table row, the condition icon doesn't appear on your page, so you must bind its attributes differently. Instead of displaying an icon, the row is shaded in blue. To create a condition around a table row:

  1. Select an item in the row, click the <TR> tag in the path view, and click the WOConditional button .

    Note that selecting every cell in the row does not select the table row (<TR>) element, but only the individual table cell (<TD>) elements. To select the row, you must click on the <TR> tag in the path view.
  2. Drag from a Boolean key to the <WOConditional> tag in the path view, and choose conditional from the menu that appears.
  3. If you want the contents displayed only if condition is false, in the Inspector set negate to true.

To display one chunk of text when a condition is true and other chunk of text when it's false:

  • Wrap each chunk of text in a conditional, bind their condition attributes to the same key, and set the negate attribute of one conditional to false and the other to true.
