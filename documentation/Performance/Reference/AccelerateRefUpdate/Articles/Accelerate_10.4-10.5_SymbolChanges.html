vImageOverwriteChannelsWithPixel_ARGB8888Overwrites an ARGB8888 image buffer with the provided pixel value.
vImageOverwriteChannelsWithPixel_ARGBFFFFOverwrites an ARGBFFFF image buffer with the provided pixel value.
