Optional Zip Drive

As an option, the Power Mac G4 can have an internal Iomega 250 MB Zip drive. It is an ATAPI drive connected as device 1 in an IDE Device 0/1 configuration on the IDE channel of the main logic board. If the Zip drive option is not installed at the time of purchase, data and power connectors are provided to add an ATAPI Zip drive to the system. The device should be device-select jumpered as device 1 (slave).
