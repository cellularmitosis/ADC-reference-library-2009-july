About This Book

This book describes the tools and techniques that system administrators and website managers perform to deploy WebObjects applications. The WebObjects Deployment package allows you to deploy applications developed with the WebObjects Development package, so that they can be accessed through a Web server. You need a WebObjects deployment license to deploy WebObjects applications

The book is intended primarily for system administrators. Application developers can also benefit from the information it provides but it's not required reading for them. This edition of the book is intended for WebObjects 5.0.
