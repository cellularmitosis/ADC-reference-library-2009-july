Modem Support

One of the USB ports on the Pangea IC is used for the interface to the modem. The Pangea IC provides DB-DMA support for the modem interface. The modem provides digital call progress signals to the Tumbler sound circuits.

The internal hardware modem is a separate module that contains the datapump and the interface to the telephone line (DAA). For more information about the modem, see "Internal Modem".
