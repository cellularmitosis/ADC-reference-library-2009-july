Accessibility (ApplicationServices/&zwjHIServices) Reference Constants, Globals, and Macros Index

The Accessibility API supports the development of applications that are accessible to assistive applications and technologies. The Accessibility header files are packaged in the HI Services subframework of the Application Services umbrella framework.
