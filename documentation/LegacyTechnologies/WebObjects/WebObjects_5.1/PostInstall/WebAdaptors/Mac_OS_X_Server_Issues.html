Mac OS X Server Issues

If there are problems executing the CGI adaptor on MacOS X Server, make sure that the "WebObjects" CGI executable is located in /Library/WebServer/CGI-Executables/, is owned by root:admin, and is executable by everyone. If MacOS X Server is running on an HFS filesystem, stop the Web server, remove the /tmp/WOAdaptorState file, "touch WebObjects", and restart the Web server. You may have to repeat this process (i.e. "touch WebObjects" and restart) after every boot or reboot for the Web server to notice it.
