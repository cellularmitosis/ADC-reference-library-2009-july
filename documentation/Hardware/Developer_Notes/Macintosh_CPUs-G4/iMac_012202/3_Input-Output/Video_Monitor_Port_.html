Video Monitor Port

The iMac (Flat Panel) computer has a port for an external video monitor. The external monitor shows the same information as the built-in display, a mode sometimes called hardware video mirroring. The external monitor supports only the display sizes that are available on the internal monitor; the sizes and refresh rates are shown in Table 3-7.


Table 3-7 Display sizes and refresh rates


Dislpay size Refresh rate
640 by 480 60 Hz
800 by 600 75 Hz
1024 by 768 75 Hz
