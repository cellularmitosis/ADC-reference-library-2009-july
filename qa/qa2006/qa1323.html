Technical Q&A QA1323
Interface element identification in Interface Builder

Q: How can I visually identify the various components of my interface in Interface Builder?

A: You can get a visual pointer to any interface element on-screen by holding down ctrl+shift while clicking an interface element in the outline view of the Instances pane. A translucent blue arrow will appear next to the element in question, as illustrated below:
