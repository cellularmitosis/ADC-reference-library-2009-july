Core Foundation
Core Foundation is a framework that provides fundamental software services useful to application services, application environments, and to applications themselves. With Core Foundation, developers can build cross-platform, server-related applications, and can share code and data among frameworks, libraries, and applications in different environments and layers. Core Foundation also provides abstractions for common data types, facilitates internationalization with Unicode string storage, and offers a suite of utilities such as plug-in support, XML property lists, URL resource access, and preferences.
