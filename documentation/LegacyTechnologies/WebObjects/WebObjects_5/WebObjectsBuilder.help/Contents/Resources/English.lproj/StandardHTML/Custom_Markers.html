Custom Markers

Not all legal HTML elements can be created directly using WebObjects Builder's buttons or menu commands. However, you can create any type of element using the custom tag.

To create an HTML element using a custom marker:

  • Choose Elements > Custom Elements or click the Custom Element button, and enter the element's name.

    If this item has no end tag, deselect "New marker is a container (allow children)."

To add attributes to this element:

  • Control-click the marker's tag, choose Inspect, and press Return.

    WebObjects Builder displays the Inspector and adds an entry to its attribute table.

    Tip: You can also edit the marker's source directly in the source view. Any attributes you add in the source view are reflected in the Inspector.

To re-use a custom marker easily, save it on a palette. See Using Palettes.
