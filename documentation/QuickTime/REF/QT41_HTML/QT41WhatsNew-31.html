AUTOHREF Embed Tag

The AUTOHREF tag causes the QuickTime Plug-in to load the URL specified in the HREF tag immediately instead of waiting for the user to click on the movie. Note that this tag modifies the HREF tag and so has no effect unless there is also an HREF tag.
