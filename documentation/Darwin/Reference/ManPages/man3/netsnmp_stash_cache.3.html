Detailed Description
       Automatically caches data for certain handlers.

       This handler caches data in an optimized way which may aleviate the need for the lower level handlers
       to perform as much optimization. Specifically, somewhere in the lower level handlers must be a
       handler that supports the MODE_GET_STASH operation. Note that the table_iterator helper supports
       this.
