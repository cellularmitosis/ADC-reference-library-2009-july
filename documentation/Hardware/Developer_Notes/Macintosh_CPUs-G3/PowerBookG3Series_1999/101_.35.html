Control Keys

The cursor control keys are also used as page control keys. Another set of keys take on the functions of keys on a PC keyboard, for use with PC emulation software. These keys are affected by the Fn key. Table 3-7 is a list of the affected keys and their alternate functions.  

Table 3-7 Control keys that change

Key name

Alternate function
Shift Right shift key
Control Right control key
Option Alt gr (right Alt key)
Command Windows ® key
Enter Menu key (for contextual menus)
Left arrow Home
Up arrow Page up
Down arrow Page down
Right arrow End
