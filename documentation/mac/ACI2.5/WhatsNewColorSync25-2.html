What's New in ColorSync 2.5

This document describes new features available with ColorSync Manager Version 2.5.

For information on features available with ColorSync Manager 2.1 and earlier versions, see the electronic bookAdvanced Color Imaging on the Mac OS , which also describes the Color Picker Manager (Version 2.0), Color Manager, and Palette Manager.

Technote 1100, "Color Picker 2.1" describes version 2.1 of the Color Picker Manager.
