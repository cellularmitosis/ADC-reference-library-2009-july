Sound IC Support

The Pangea IC has a traditional DAV port that connects to the Screamer sound IC. The Pangea IC provides DB-DMA support for the DAV port.

The Screamer sound IC is an audio codec that provides analog input conditioning, A/D (analog to digital) conversion, D/A (digital to analog) conversion, and analog output conditioning. The A/D and D/A conversions use 16-bit data.

For a description of the features of the sound system, see "Sound System".
