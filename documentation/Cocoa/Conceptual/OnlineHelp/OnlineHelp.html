Introduction to Online Help

Contents:

Organization of This Document


This programming topic describes the various types of on-line help found in Mac OS X. Your application rarely needs to contain code to display on-line help. Instead, you use Interface Builder and Xcode to set up on-line help for your application.

Organization of This Document

This programming topic contains the following articles:

  • “Comprehensive Help” describes the comprehensive help you can provide.

  • “Tooltips” describes the short bits of text that can describe a user interface element.

  • “Specifying the Comprehensive Help File” describes how to specify the location for your comprehensive help.
