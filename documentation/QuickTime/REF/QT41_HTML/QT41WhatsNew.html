What's New in QuickTime 4.1

Overview

This document provides you with a list of changes from QuickTime 4 to QuickTime 4.1, as well as new features and capabilities available in this software release. It is intended to supplement the information provided in the QuickTime 4 API Reference.

If you are a QuickTime API-level developer or a QuickTime content creator or provider, you should read this document.
