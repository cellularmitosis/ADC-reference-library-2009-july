iChatStatusFromApplication

Version: 1.1

Posted: 2008-03-31

Build Requirements: Xcode Tools 3.0 and Mac OS X version 10.5

Runtime Requirements: Mac OS X version 10.5

View Source Code:

Download Sample (“iChatStatusFromApplication.zip”, 105.2K)
Download Sample (“iChatStatusFromApplication.dmg”, 161.3K)


Description

This sample illustrates how to use the Scripting Bridge framework to communicate with iChat. It watches for frontmost-application changes, and updates the user's status message and buddy icon.
