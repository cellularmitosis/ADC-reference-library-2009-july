Changes to QuickTime for Java

There are a number of changes in this release of QuickTime for Java. The primary changes include:

  • Deprecation of AWT dependencies

  • The addition of two new packages in the app tree:

    • quicktime.app.event -- a new event model for QuickTime for Java classes

    • quicktime.app.ui -- which includes a button sprite

For general information about QuickTime for Java, refer to

<http://www.apple.com/quicktime/qtjava/>.

A summary of the QuickTime for Java API is available at

<http://developer.apple.com/documentation/quicktime/qtdevdocs/Java/tp_java_digest.htm>.
