Manager: ADB Manager


Carbon does not support the ADB Manager, because it accesses hardware directly. As with other hardware-specific managers, you can use this manager through a non-Carbon plug-in for Mac OS 8. For Mac OS X, you should use the comparable APIs supplied with the IOKit.
