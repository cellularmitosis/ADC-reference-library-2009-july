WOSMILDocument

This is the root element of every SMIL component in WebObjects. It corresponds to the <smil> tag.


WOSMILHead

This element corresponds to the <head> tag.


WOSMILBody

This element defines how elements are rendered in the component's timeline. It corresponds to the <body> tag. It normally encloses WOSMILPar, WOSMILSeq, or WOSMILSwitch elements.
