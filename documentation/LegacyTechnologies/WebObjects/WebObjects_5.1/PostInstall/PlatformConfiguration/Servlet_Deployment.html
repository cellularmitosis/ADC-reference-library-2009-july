Servlet Deployment

If you Installed WebObjects Deployment with Servlet deployment (a custom installation) the following notes apply to your installation.

Configure your application server to pass on the following Java properties:

  • Mac OS X
    • -DWOROOT=/ -DLOCALROOT=/Local
    • -DWOAINSTALLOOT=/Local/Library/WebObjects/Applications
  • Solaris
    • -DWOROOT=/opt/Apple -DLOCALROOT=/opt/Apple/Local
    • -DWOAINSTALLOOT=/opt/Apple/Local/Library/WebObjects/Applications
  • Windows
    • -DWOROOT=C:/Apple -DLOCALROOT=C:/Apple/Local
    • -DWOAINSTALLOOT=C:/Apple/Local/Library/WebObjects/Applications

Place your applications in NEXT_ROOT/Local/Library/WebObjects/Applications.

Place your frameworks in NEXT_ROOT/Local/Library/Frameworks.
