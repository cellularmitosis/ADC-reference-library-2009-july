Technical Q&A AMTPE20
Building an Application for Windows in Apple Media Tool

The Apple Media Tool and Apple Media Tool Programming Environment products have been discontinued. For more information check out: AMT/PE Discontinued.

Q: We would like to see an application that has been converted to run in the Windows environment before investing in Apple Media Kit. Is it possible to build a stand-alone application from the Apple Media Kit demo?

A: The demo does allow you to create an Apple Media Tool project and to save that project file for later compilation by the complete version of the Apple Media Tool. Although you can run your project within the demo application, you cannot see how it looks on the PC unless you have the full version. However, the transfer to the PC is relatively easy, and what you see on the Mac is the same as what you will see on the PC.

You can check out any of the titles mentioned in the titles list on the demo CD. Most are cross-platform and available in computer stores.

[Aug 01 1995]
