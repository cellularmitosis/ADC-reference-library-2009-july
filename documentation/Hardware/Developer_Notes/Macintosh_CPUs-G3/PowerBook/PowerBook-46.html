FireWire Device Programming

Developers of FireWire peripherals are required to provide device drivers. A driver for DV (digital video) is included in QuickTime 4.0.

The PowerBook computer can boot from a FireWire storage device that implements SBP-2 (Serial Bus Protocol) with the RBC (reduced block commands) command set. Detailed information is available only under non-disclosure agreement; contact Developer Technical Support at dts@apple.com.

When connected to another computer by a FireWire bus, the PowerBook computer can operate as a mass storage device. See Target Disk Mode.

For additional information about the FireWire interface and the Apple APIs for FireWire device control, refer to the resources available on the Apple FireWire web site at

http://developer.apple.com/hardware/FireWire/

or send electronic mail to

firewire@apple.com
