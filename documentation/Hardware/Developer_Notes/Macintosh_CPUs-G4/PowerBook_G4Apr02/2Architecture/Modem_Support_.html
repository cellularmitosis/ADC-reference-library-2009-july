Modem Support

Unlike earlier Macintosh models which used serial-port modems, the internal modem is connected to an internal USB port. The KeyLargo IC provides DB-DMA support for the modem interface. The modem provides digital call progress signals to the Snapper sound circuitry.

The internal modem is a separate module that contains the datapump IC and the interface to the telephone line (DAA). For more information about the modem, see "Internal Modem".
