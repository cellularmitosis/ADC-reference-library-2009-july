Hardware Graphics Acceleration

Mac OS X version 10.2 incorporates a new system for turning on hardware graphics acceleration in you Java Swing applications. Feedback from developers has encouraged the change from a policy where you must explicitly state on which video cards hardware acceleration should be turned on to a policy where you only need to designate which cards you do not want hardware acceleration turned on for.
