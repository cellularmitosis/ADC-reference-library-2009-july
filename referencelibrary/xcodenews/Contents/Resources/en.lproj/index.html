Project
  • Project Snapshots

    Experiment without fear. Record all the files of your project at any point in time with the confidence that you can always restore your project to a past working state. To take a snapshot, select File > Make Snapshot. To retrieve your Snapshot files, select File > Snapshot.

  • Organizer Window

    Develop your next project more easily, regardless of language and without a project. Just drag a folder into the Xcode Organizer window and click Build.

Editing
  • Open Quickly

    To open a file whose name appears in the text editor, select the name and choose: File > Open Quickly. To open a file by typing its name, open the Open Quickly dialog and enter the filename in the Path field.

  • Edit all in scope

    To rename a variable everywhere it occurs in the current scope, select the variable name and choose Edit > Edit all in Scope.

  • Refactoring

    Refactoring allows you to improve the readability of a product’s source code while retaining its functionality and behavior.

Documentation
  • Symbol documentation

    Option-double-click a symbol in an editor to be taken to the documentation for that symbol.

  • Header file

    Go directly to the header file of a symbol by Command-double-clicking it.

  • Research Assistant

    Help > Show Research Assistant brings up the Research Assistant for live symbol information as you code. The Research Assistant can also be accessed from the Help menu.

  • Documentation window

    Browse the documentation in the documentation window, accessed with Option-Command-?. The documentation window can also be accessed from the Help menu.

  • Keep Your Library Up to Date

    Enable automatic updating in Xcode to get the latest iPhone Reference Library documentation.

Debugging
  • Vertical debugging

    Change the layout of the debugger window to a vertical layout by choosing: Run > Debugger Display > Vertical Layout

  • Filing a bug with Apple

    Apple hosts a public site knows as Bug Reporter so you can submit bugs you find in the tools or any other Apple software.
