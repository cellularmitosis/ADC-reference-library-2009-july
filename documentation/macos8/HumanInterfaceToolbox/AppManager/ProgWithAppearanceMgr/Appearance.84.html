Programming With the Appearance Manager

Data Types

The Appearance Manager supplies the following data types:

  • ThemeDrawingState
  • ThemeButtonDrawInfo
  • ThemeTrackDrawInfo
  • ScrollBarTrackInfo
  • SliderTrackInfo
  • ProgressTrackInfo
  • ThemeWindowMetrics
  • ThemeIteratorUPP
  • ThemeEraseUPP
  • ThemeTabTitleDrawUPP
  • ThemeButtonDrawUPP
  • WindowTitleDrawingUPP
  • MenuTitleDrawingUPP
  • MenuItemDrawingUPP
