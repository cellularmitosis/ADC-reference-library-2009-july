D2WCollapsibleComponentContent

Package:

Inherits from:com.apple.yellow.webobjects.WOComponent

Class Description

This reusable component is an enhancement to the WOCollapsibleComponentContent reusable component in the WebObjects Extensions Framework. It differs from WOCollapsibleComponentContent because it prevents the collapse and expand icons from appearing on different lines than the content.
