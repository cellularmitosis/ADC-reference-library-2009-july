Description

Reminders demonstrates how you can use a WebKit plug-in to extend the functionality of a Dashboard widget. It shows how to access the CalendarStore framework from a WebKit plug-in, update and retrieve widget preferences, and call JavaScript from a WebKit plug-in and vice versa.

It is a Dashboard widget that displays upcoming iCal events and to do items occurring within the year.
