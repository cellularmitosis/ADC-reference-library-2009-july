PowerStep

PowerStep is a feature that allows the user to change the processor's clock speed, either slowing down to 400 MHz to conserve power or speeding up when the full 500-MHz speed is needed. The user can select this feature by using either the Energy Saver control panel or the Energy Saver control strip. In the Energy Saver control panel, the user can check a check box labeled Reduce Processor Speed in the Advanced Settings panel. In the Energy Saver control strip, the user can select Faster Processor Speed or Slower Processor Speed.
