Unused IDE Signals on the Expansion Bay Connector

Several signals defined in the standard interface for the IDE drive are not used by the expansion bay. Those signals are listed in Table 4-4 along with any action required for the device to operate in the expansion bay.

Table 4-4 Unused IDE signals on the expansion bay connector

Signal name

Comment
CSEL This signal must be tied to ground to configure the device as the master in the default mode.
PDIAG No action required; the device is never operated in master-slave mode.
DAS No action required.
