Hardware Features

Table 1-1 provides a list of the hardware features of the Xserve slot load and cluster node configuration computers. Each of the major features is described more fully later in this note, as indicated by the cross references.

Table 1-1 Hardware Features of the Slot Load and Cluster Node Xserve


Feature Slot Load Xserve Cluster Node Xserve
microprocessor speed Dual or single PowerPC G4 microprocessor running at a clock frequency of 1.33 GHz. See “PowerPC G4 Microprocessor”. Dual PowerPC G4 microprocessor running at a clock frequency of 1.33 GHz. See “PowerPC G4 Microprocessor”.
Memory caches Internal 256 KB level 2 cache per processor and external 2 MB level 3 cache per processor. See “Cache Memory”.
Processor system bus 64-bit wide data and 32-bit wide address, 167 MHz clock, supporting MaxBus protocol. See “Processor Bus”.
RAM Four DIMM slots for 184-pin DIMMs (dual inline memory modules) using DDR (double data rate) dynamic RAM devices. A minimum of 256 MB of RAM is installed in one of the slots. See “RAM Expansion”.
ROM ROM-in-RAM implementation with 1 MB of boot ROM. For information about the ROM, see “Boot ROM”. For information about the ROM-in-RAM implementation, see the references listed in “ROM-in-RAM Architecture”.
Graphics card PCI or AGP graphics card with a VGA connector for the monitor. See “Graphics Cards”. No graphics card.
Hard disk drive bays Four drive bays for Ultra DMA/133 with independent buses and support for hot-pluggable drives using Apple Drive Modules. See “Hard Disk Drive Bays”. One drive bay for Ultra DMA/133 with independent bus and support for hot-pluggable drive using Apple Drive Modules. See “Hard Disk Drive Bays”.
CD-ROM drive Contains CD-ROM drive. See “CD-ROM Drive”. Not available.
Combo drive A CD-RW/DVD-ROM Combo drive is available as a build-to-order option. See “Combo (DVD-ROM/CD-RW) Drive (Optional)”. Not available.
Hard disk 7200-rpm, hot-pluggable, 60 GB hard disk drive in one of the four drive bays. 180 GB build-to-order drive available. 7200-rpm, hot-pluggable, 60 GB hard disk drive.
USB ports Two USB ports. See “USB Ports”.
Ethernet Two Ethernet ports for 10Base-T, 100Base-T, or 1000Base-T operation. One port is on the main logic board and one is on a network card in a slot. See “Ethernet Ports”. One Ethernet ports for 10Base-T, 100Base-T, or 1000Base-T operation on the main logic board. See “Ethernet Ports”.
FireWire ports Two FireWire 800 ports on back panel and one FireWire 400 on front. See “FireWire Ports”.
PCI card expansion slots Two expansion slots for PCI cards. See “PCI Expansion Slots”.
PCI/AGP card slot One half-length PCI or AGP slot for a graphics or networking card. See “PCI Expansion Slots”. No AGP slot available.
Fan speed control The system monitors the fan speeds and reports if speeds are outside acceptable range, indicating that a fan needs service.
