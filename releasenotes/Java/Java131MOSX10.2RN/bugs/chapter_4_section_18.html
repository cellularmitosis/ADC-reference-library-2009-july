Java Widgets

Radar #2877462

Textfield Layout and Event Problems after Java 1.3.1 Update 1.

Description:

After installation of the Java 1.3.1 Update 1 some layout problems have been reported when using text fields in a modal dialog, when contained inside a lightweight container.

Workaround:

Use heavyweight containers for AWT text components.
