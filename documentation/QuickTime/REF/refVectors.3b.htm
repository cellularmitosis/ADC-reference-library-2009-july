Specifying a Transfer Mode

You specify a transfer mode by including a kCurveTransferModeAtom atom, described in "Vector QT Atom Container" , in a vector data stream. By default, no transfer mode is used. For descriptions of transfer modes, see "Transfer Modes" . The data for a kCurveTransferModeAtom atom is a gxTransferMode structure, which is described in "The Transfer Mode Structure" .
