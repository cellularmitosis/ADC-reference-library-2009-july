Java Borland External

Radar #2800247

Crash when trying to evaluate a call with an array parameter using a debugger.

Description:

Java's method signature parser for evaluation fails when the arguments include an array of arrays or an array of built-ins. Only arrays of objects work.

Workaround:

This has been fixed.
