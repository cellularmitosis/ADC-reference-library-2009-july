Inside Macintosh: Advanced Color Imaging

Archives of the Original Advanced Color Imaging Documentation

From this page you can obtain the orginal versions of Advanced Color Imaging on the Mac OS and Advanced Color Imaging Reference. These are published by Addison Wesley as a retail product that combines a printed book, Advanced Color Imaging on the Mac OS, with a CD-ROM disk containing an electronic book, Advanced Color Imaging Reference.

The printed book, Advanced Color Imaging on the Mac OS, describes how to use the Palette Manager, the Color Picker Manager (version 2.0), the ColorSync Manager (version 2.0), and the Color Manager. The accompanying CD-ROM disk contains Acrobat and QuickView formats of Advanced Color Imaging Reference, which in turn document the constants, data types, and functions defined by the Palette Manager, the Color Picker Manager (version 2.0), the ColorSync Manager (version 2.0), and the Color Manager.
