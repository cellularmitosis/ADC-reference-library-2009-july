ATA Manager Reference

This chapter defines the data structures and functions used by the ATA Manager. The section The ATA Parameter Block shows the data structure of the ATA parameter block. Version 3.0 and later of the ATA Manager support DMA data transfers. The section Setting Data Transfer Timing discusses how the ATA Manager interacts with ATA devices to setup DMA transfers. The Functions section describes the functions for managing and performing data transfers through the ATA Manager.
