Manager: Cursor Devices


The functionality provided by the Cursor Device Manager is specific to Apple Desktop Bus (ADB) devices and therefore is not supported in Carbon. You should instead use Input Sprockets for compatibility with both ADB and USB devices on Mac OS 8 and Mac OS 9. For Mac OS X, IOKit provides methods of accepting user input.
