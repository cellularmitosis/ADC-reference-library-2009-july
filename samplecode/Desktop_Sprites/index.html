Description

Demonstrates how to create a Sprite World and it's associated sprites. Also demonstrates how to animate sprites by modifying sprite properties. Part of An Introduction to QuickTime. Requirements: A Power Macintosh Computer running Mac OS 9 (9.2.2 or later recommended), Metrowerks CodeWarrior Pro version 7.0 or later, QuickTime 5.0.2, CarbonLib 1.5 and Universal Interfaces 3.3.4 or later. A Power Macintosh Computer running Mac OS X (10.1.2 or later recommended) with Mac OS X Developer Tools December 2001 or later. Keywords: QuickTime, Sprites
