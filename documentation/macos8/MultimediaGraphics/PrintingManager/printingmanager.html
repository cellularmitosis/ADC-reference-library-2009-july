PATH Documentation > Mac OS 8 and 9 > Multimedia and Graphics > Printing Manager
The Printing Manager is a collection of system software routines that support printing in Classic applications. This manager has been replaced by the Carbon Printing Manager, a Carbon-compliant API that allows applications to print both on Mac OS 8 and 9 with existing printer drivers and on Mac OS X with new printer drivers.
