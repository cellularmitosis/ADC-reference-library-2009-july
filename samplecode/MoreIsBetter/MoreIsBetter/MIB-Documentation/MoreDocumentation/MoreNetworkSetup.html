MoreNetworkSetup

MoreNetworkSetup contains two key modules:

  • MoreNetworkSetup -- A simple wrapper around the Network Setup library that makes it somewhat easier to use.
  • NetworkSetupHelpers -- All the code you need to write an Internet setup utility for Mac OS. This module works with both the Network Setup library (Mas OS 8.5 and higher) and pre-8.5 versions of Open Transport.

The entire module has significant documentation.

MoreNetworkSetup supports 68K and PowerPC development; it does not support Carbon development.
