Leopard Sample Code: QuickTime
Java
QuickTime includes a rich set of multimedia classes and methods for Java programmers, supporting creation of cross-platform applications for Mac OS and Windows. Supported media include video, sound, text, animation, graphics, and more. Supported formats include MPEG-4 video and AAC audio. These applications can be written on Mac OS X using Xcode and turned into double-clickable Mac OS X applications using Jar Bundler.
