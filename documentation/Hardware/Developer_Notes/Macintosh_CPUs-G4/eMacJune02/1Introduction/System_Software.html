System Software

The eMac computer comes with both Mac OS X (version 10.1.4) and Mac OS 9.2 installed. Mac OS X is the default system. (A configuration with Mac OS 9 as the default system is also available for education customers.)

For more information about Mac OS X, see the reference listed in "Mac OS X". For the latest information about Mac OS 9.2, see the reference listed in "Mac OS 9.2".
