Manager Group: Networking and Communication Services


In Carbon, Open Transport replaces most other commonly-called Mac OS programming interfaces for network and communication services. With the exception of Open Transport, the URL Access Manager, Internet Config and the Network Services Location Manager, all previous networking and communication managers are unsupported in Carbon.
