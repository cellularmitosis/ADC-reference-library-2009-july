EventMonitorTest

Version: 1.0

Posted: 2005-11-09

Build Requirements: XCode 2.1 or greater on OS X 10.4.x. The sample was designed for use on OS X 10.3.x. One could create a project using Xcode 1.5 under OS X 10.3.x.

Runtime Requirements: The sample requires OS X 10.4.x. The sample code was originally written for OS X 10.3.x.

Download Sample (“EventMonitorTest.zip”, 36.0K)
Download Sample (“EventMonitorTest.dmg”, 92.5K)


Description

Sample to demonstrate the use of the GetEventMonitorTarget for installing an event notifier of all keyboard, mouse and tablet events while a process is in the background. The sample builds in Universal Binary format.
