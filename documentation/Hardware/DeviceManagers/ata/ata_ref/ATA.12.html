verify

The verify control function performs no operation and returns noErr if the parameters passed in the call are valid.

An arrow preceding a parameter indicates whether the parameter is an input parameter, an output parameter, or both.

Meaning
--> Input
<-- Output
<--> Both


Parameter block

-->

csCode

-->

ioVRefNum

 The logical drive number.

csParam[ ]

 See result codes.
