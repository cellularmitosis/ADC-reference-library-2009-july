Begin Group Objects

Labels

ASCII
BeginGroup
Binary
bgng ( = 0x62676E67 )

Description

A begin group object is used to declare a group and to delimit the start of that group. Every group must begin with a begin group object.

Parent Hierarchy

3DMF.

Parent Objects

None.

Child Objects

None.

Example

BeginGroup(
    DisplayGroup ( )                    # empty group
)
EndGroup ( )
