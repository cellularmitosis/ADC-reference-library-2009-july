Text Regions

A Text Region contains a block of text that can be any length.

To add a text region to a form:

  • Click the Text Region button , and then bind the field's value attribute to the variable that holds the user's data.

To set the region's length:

  • In the Static Inspector, use the Size fields.

    "columns" is the length of the region in characters. "rows" is the height of the region in text lines.
