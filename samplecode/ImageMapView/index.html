Description

Shows how to handle Accessibility Carbon Events in a custom HIView. Accessorizes subcomponents of a custom HIView. Shows how to add auxiliary Acessibility atrributes (AXDescription and AXTitleUIElement). Shows how to mark a given HIView as Accessibility ignored.

This sample code has been updated to include a project that produces a universal binary. No code changes were required for it to run correctly on the Developer Transition Systems.
