Keyboard Layout

There are versions of the Apple USB keyboard with different key layouts for three standards used in different parts of the world: ANSI (US and North America), JIS (Japan), and ISO (Europe). Figure 3-4 shows the keyboard layout for the ANSI keyboard. Applications can determine which keyboard is connected by calling the Gestalt Manager and checking for the corresponding value of the gestaltKeyboardType selector:

  • gestaltUSBCosmoANSIKbd (value = 198)

  • gestaltUSBCosmoJISKbd (value = 200)

  • gestaltUSBCosmoISOKbd (value = 199)
