Overview

Important: The information in this document is obsolete and should not be used for new development.

An NSActionCell defines an active area inside a control (an instance of NSControl or one of its subclasses). As an NSControlâ€™s active area, an NSActionCell does three things: it usually performs display of text or an icon; it provides the NSControl with a target and an action; and it handles mouse (cursor) tracking by properly highlighting its area and sending action messages to its target based on cursor movement.
