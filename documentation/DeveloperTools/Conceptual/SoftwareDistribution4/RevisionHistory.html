Legacy Document

Important: The information in this document is obsolete and should not be used for new development.

Document Revision History

This table describes the changes to Software Distribution Legacy Guide.

DateNotes
2006-07-24Made into legacy document.
 

In “Keys That Can Appear in Any Package,” added note that Installer ignores any values supplied for the IFMajorVersion and IFMinorVersion keys. (See Installer Release Notes for the latest information on changes in Mac OS X v10.4.)

 

In same section, updated the description for the CFBundleIdentifier key.

 

“Search Methods” now describes what happens if you use the Find File mechanism and the user has more than one copy of the software to be updated.

 

“Getting Software to the User” now distinguishes between the Disk Utility and Disk Copy applications and explains how to view the Disk Utility Help.

 

“Distributing Software With Internet-Enabled Disk Images” notes that Disk Copy was merged with Disk Utility in Mac OS X version 10.3.

 

In description of IFPkgFlagInstallFat flag, noted that with the default value of NO, .o or .a files installed with a package will be thinned along with any other binaries (in “Keys That Can Appear Only in Single Packages”).

 

In description of variables for installation check files, noted that the values for the variables $2, $3, and $4 cannot be trusted (in “Arguments and Environment Variables”).

2004-05-27

Modified descriptions (in several places) of when a user can choose a location for packages in a metapackage. See, for example, “Choosing the Location for a Metapackage.”

 

Added the section “Modify the Package for Special Processing.” It includes some information previously in another section, plus a new link to “Specifying Installation Requirements.”

 

Added the section “Obtaining Owner and Group Information.” This covers the information from Technical Q&A QA1285.

 

Added the trouble-shooting section “Why does my package fail in Mac OS X version 10.1 but not in version 10.2?.” This covers the information from Technical Q&A QA1283.

 

Added the trouble-shooting section “Why is Installer ignoring the version.plist information in my package?.” This covers the information from Technical Q&A QA1281.

 

Made minor spelling corrections.

2003-08-21

Updated documentation to reflect new and revised features in Mac OS X version 10.3 (v10.3).

 

Added “Specifying Installation Requirements,” which describes a new mechanism available in Mac OS X v10.3 for specifying installation requirements.

 

Added a list of supported localization folder names in “Localized Folder Names” (not specific to Mac OS X v10.3).

 

Revised “Authorization, File Ownership, and Permissions,” which was formerly called “Authorization and File Ownership”.

 

Now includes additional information on default ownership and permissions.

 

Updated “Finding Previously Installed Software”:

 

Added descriptions for new search methods:

 

“BundleVersionFilter”

 

“CommonAppSearch”

 

Added examples and updated various descriptions.

 

Updated “PackageMaker and Installer Features” for new and changed features in Mac OS X v10.3.

 

Modified “Other Keys You Might Want to Modify” to list new options for:

 

specifying installation requirements by modifying a package’s information property list;

 

specifying that an installation requires a logout by modifying a package’s information property list

2003-05-13

First version of Software Distribution released.
