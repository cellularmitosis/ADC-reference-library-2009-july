System RAM

The memory subsystem in the iMac family of computers supports two slots for 168-pin DIMMs (dual inline memory modules). One slot is occupied by a DIMM containing 64 MB of SDRAM; the second slot is an expansion slot. The data bus to the RAM and DIMM is 64 bits wide, and the memory interface is synchronized to the 60x bus interface at 100 MHz. See RAM Expansion.
