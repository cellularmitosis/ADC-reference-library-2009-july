Programming With the Mac OS 8.5 Window Manager

IsWindowModified

Obtains the modification state of the specified window.

pascal Boolean IsWindowModified (WindowPtr window);
window
A value of type WindowPtr . Pass a pointer to the window whose modification state is to be obtained.
function result
A value of type Boolean . The function returns true to indicate that the content of the window has been modified; otherwise, false . Newly created windows start out with their modification state automatically set to true .
DISCUSSION

Your application can use the functions IsWindowModified and SetWindowModified instead of maintaining its own separate record of the modification state of the content of a window.

VERSION NOTES

Available with Mac OS 8.5 and later.
