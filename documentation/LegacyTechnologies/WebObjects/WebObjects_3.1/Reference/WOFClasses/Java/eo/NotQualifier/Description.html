Class next.eo.NotQualifier

CLASS DESCRIPTION

Extends:
next.eo.Qualifier

NotQualifier is a subclass of Qualifier that contains a single qualifier. When a NotQualifier object is evaluated, it returns the inverse of the result obtained by evaluating the qualifier it contains.

CONSTRUCTORS

NotQualifier

public NotQualifier()
public NotQualifier(next.eo.Qualifier aQualifier)

Returns a newly-allocated and initialized NotQualifier object. If aQualifier is supplied, this method initializes the receiver with the Qualifier aQualifier.
