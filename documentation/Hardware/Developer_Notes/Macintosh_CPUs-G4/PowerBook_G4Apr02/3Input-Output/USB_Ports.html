USB Ports

The PowerBook G4 computer has two external USB 1.1 ports that can be used to connect additional I/O devices such as a USB mouse, printers, scanners, and low-speed storage devices.

The USB ports on the PowerBook G4 computer comply with the Universal Serial Bus Specification 1.1 Final Draft Revision. For more information about USB on the Macintosh computer, consult the references at "USB Interface".
