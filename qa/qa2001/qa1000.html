Technical Q&A QA1000
Gathering system information under Traditional Mac OS

Q: I want to be able to collect information about the machine like Apple System Profiler does. How do I do this? Is the source code for Apple System Profiler available?

A: Most of the information displayed by Apple System Profiler can be found using the Gestalt Manager and the Name Registry.

The source code for Apple System Profiler is not available.

[Jan 10 2001]
