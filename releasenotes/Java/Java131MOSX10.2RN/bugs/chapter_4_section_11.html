Java Internet Plugin

Radar #2968498

IE doesn't support Java Plugin feature.

Description:

The current version of Internet Explorer included in Mac OS X 10.2 doesn't support the Java Applet.plugin feature as described in What's New in Java on Mac OS X Version 10.2.

Workaround:

None at this time.
