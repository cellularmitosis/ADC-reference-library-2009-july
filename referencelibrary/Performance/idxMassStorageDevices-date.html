Leopard Reference Library: Performance
Storage
Storage devices include disk drives, tape drives, RAID arrays, and Storage Area Network (SAN) file systems. Xsan administrators can tune storage devices using the Xsan administrative tools. Developers can fine-tune their applications by measuring and evaluating file-system performance using tools such as Shark and fs_usage.
