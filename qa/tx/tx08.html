Technical Q&A TX08
Using the Licensed Espy Font

Q Is the Espy font family, found in AppleGuide, freely distributable? It's such a nice screen font, we'd like to use it in a commercial application that we are going to release; but we obviously don't want to violate any license restrictions.

A Apple designed the Espy font family as a new, more readable screen font, initially used for the Newton's main text face. It has since migrated to various other apps, such as AppleGuide. Unfortunately, the font cannot be licensed. The only advice we can offer is to check with font developers such as Adobe or Bitstream to see if they have a similar typeface that you can license.

[Oct 25 1996]
