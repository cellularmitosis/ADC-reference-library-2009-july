URLs in Text Tracks

In QuickTime 4.1, you can have URL links in text tracks, as shown in the example in Listing 5. This example contains a number of links which, once saved as a text file, can be imported into QuickTime Player.

Note that these URLs in text tracks cannot have line breaks in them, though they are printed that way to fit on a page here.
