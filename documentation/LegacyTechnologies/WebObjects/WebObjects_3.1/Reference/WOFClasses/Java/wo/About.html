About This Documentation

NeXT's Technical Publications group has developed this frame-based system for reference documentation. Here are some tips for using this system:
  • Long method names may wrap in the left table-of-contents frame. To fix this, make the frame wider.

  • You can easily print the complete specification for a class or interface. Just make sure the content frame is selected before you print.

  • The Examples link is inactive. We plan to have links to "live" examples sometime in the near future.

  • Longer or more important classes have "Class at a Glance" sections. Minor classes do not have these sections.

Note: This documentation package covers only the server-side Java classes related to WebObjects. It does not cover the client-side component classes in next.wo.client, next.wo.client.controls, and next.util. For documentation on those classes, see:

  • Client-Side Support Classes
  • Applet Controls

We are interested in your comments about this method of presenting reference documentation. Please send them to techpubs_feedback@next.com.
