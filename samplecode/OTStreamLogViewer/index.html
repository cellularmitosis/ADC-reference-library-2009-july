Description

OTStreamLogViewer is both a sample and a developer tool. As a sample, it shows how to use the OT raw streams API to communication with modules in the OT kernel. As a tool, it allows you to view the output of the STREAMS logging facility (strlog). This is extremely helpful when developing OT kernel plug-ins: modules drivers, and port scanners. Open Transport 1.1.1 or higher, PowerPC Open Transport kernel log strlog debug raw streams
