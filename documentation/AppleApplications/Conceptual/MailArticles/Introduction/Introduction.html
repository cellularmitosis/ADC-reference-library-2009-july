Organization of This Document


This document describes how to customize some of the features in Mail. Even though Mail does not contain an API, some features can be tailored using property lists and HTML. The articles that follow explain how to format these property list files and where to install them.

Note: This document is a series of related articles intended to be read in any order. The articles are being published in serial fashion—new articles will be added periodically.

This document is intended for developers who want to add to the functionality of Mail, or for system administrators who want to control the user’s setup process for Mail.

Organization of This Document

This document contains the following articles:

  • “Creating Mail Account Bundles” explains how to create an bundle to set up mail accounts.

  • “Creating Mail Stationery Bundles” explains how to create a stationery template in Mail.
