Class next.eo.DatabaseDataSource

CLASS DESCRIPTION

Extends:
next.eo.DataSource

DatabaseDataSource is a concrete subclass of DataSource that fetches objects based on an EOModel, using its EditingContext. A DatabaseDataSource can be set up to fetch all objects for its root entity, to fetch objects matching a particular FetchSpecification, and to further filter its fetching with an auxiliary Qualifier.

DatabaseDataSource implements all the functionality defined by DataSource. In addition to fetching objects, it can insert and delete them (provided the entity isn't read-only), can provide a detail data source, and can be set up in a master-peer configuration. See the DataSource class specification for more information on these topics.

CONSTRUCTORS

DatabaseDataSource

public DatabaseDataSource()
public DatabaseDataSource(next.eo.EditingContext anEditingContext, java.lang.String anEntityName)

Returns a newly-allocated and initialized DatabaseDataSource object. If the parameters are supplied, this method initializes the newly allocated DatabaseDataSource to fetch objects into anEditingContext for the EOEntity named by anEntityName. This method checks anEditingContext's ObjectStoreCoordinator for an EODatabaseChannel that services the EOModel containing the named EOEntity. If none exists, this method creates one. This is the designated initializer for the DatabaseDataSource class.

METHODS

auxiliaryQualifier

public next.eo.Qualifier auxiliaryQualifier()

Returns the Qualifier used to further filter the objects fetched by the receiver's FetchSpecification.

fetchSpecification

public next.eo.FetchSpecification fetchSpecification()

Returns the receiver's basic FetchSpecification. Its Qualifier is conjoined with the receiver's auxiliary Qualifier when the receiver fetches objects. The sender of this message can alter the FetchSpecification directly, or replace it using setFetchSpecification.

fetchSpecificationForFetch

public next.eo.FetchSpecification fetchSpecificationForFetch()

Returns a copy of the FetchSpecification that the receiver uses to fetch. This is constructed by conjoining the Qualifier of the receiver's FetchSpecification with its auxiliary Qualifier. Modifying the returned FetchSpecification doesn't affect the receiver's fetching behavior; use setFetchSpecification and setAuxiliaryQualifier for that purpose.

setAuxiliaryQualifier

public void setAuxiliaryQualifier(next.eo.Qualifier aQualifier)

Sets the Qualifier used to further filter the objects fetched by the receiver's FetchSpecification to aQualifier. This is useful for altering the basic FetchSpecification.

setFetchSpecification

public void setFetchSpecification(next.eo.FetchSpecification aFetchSpecification)

Sets the receiver's basic FetchSpecification to aFetchSpecification. Its Qualifier is conjoined with the receiver's auxiliary Qualifier when the receiver fetches objects.
