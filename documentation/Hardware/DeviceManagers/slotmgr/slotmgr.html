ADC Home > Documentation > Hardware > Device Managers and Drivers > Slot Manager
 
The Slot Manager allows an application or a device driver to identify NuBus expansion cards and communicate with the firmware on a card. You need to use the Slot Manager only if you are writing an application or a device driver that must address a NuBus expansion card directly.

Note: For PCI-based Macintosh computers, the Name Registry replaces the Slot Manager for most operations.
