Inspecting Objects

The Inspector lets you set HTML attributes of the elements in your component. WebObjects Builder has only one Inspector window. When you choose another object, the contents of the Inspector change.

To open the Inspector:

  • Click .

Most dynamic elements have static HTML counterparts (with the exception of abstract dynamic elements, such as: WOString, WORepetition, WOConditional, and WOCustom.) The Inspector for these elements has two parts:

  • The Dynamic Inspector lets you set the bindings for the element (see Binding Elements).
  • The Static Inspector lets you set the HTML attributes for the element's static counterpart.

Note that you can also set HTML attributes with the Dynamic Inspector.

To switch between the two parts of the Inspector, select Dynamic Inspector or Static Inspector from the pop-up menu on the Inspector's upper left side.
