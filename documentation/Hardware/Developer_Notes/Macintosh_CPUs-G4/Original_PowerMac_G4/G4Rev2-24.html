Ethernet Controller

The Uni-N IC includes an Ethernet media access controller (MAC). As a separate I/O channel on the Uni-N IC, it can operate at its full capacity without degrading the performance of other peripheral devices. The Uni-N IC provides DB-DMA support for the Ethernet interface.

The MAC implements the link layer. It is connected to a PHY interface IC that is capable of operating in either 10-BaseT or 100-BaseT mode. The actual speed of the link is automatically negotiated by the PHY and the bridge or router to which it is connected. For more information, see Ethernet Port.
