Backside Cache

The data storage for the backside L2 cache consists of 512 KB of fast static RAM on the main logic board. The controller and tag storage for the backside cache are built into the microprocessor. The cache controller includes bus management and control hardware that allows the cache to run at a sub-multiple of the processor's clock speed, rather than at the clock speed of the main system bus. The ratio of the clock speeds of the microprocessor and the backside cache is 5:2.
