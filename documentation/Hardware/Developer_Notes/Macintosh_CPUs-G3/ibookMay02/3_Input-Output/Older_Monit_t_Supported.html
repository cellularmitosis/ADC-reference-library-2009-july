Older Monitors Not Supported

The computer supports current video monitors. Some older monitors are not supported, including the following Apple monitors:

  • Multiple Scan 17
  • Multiple Scan 20
  • AudioVision 14
  • Apple Hi-Res RGB
  • Apple 16" Color
  • Apple Hi-Res Monochrome
  • Macintosh 12" RGB
