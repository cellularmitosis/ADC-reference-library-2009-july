Technical Q&A QD12
Selecting a PPD When no Printer is Available

Q: We are writing an application which has a somewhat unusual requirement: The user must be able to use the LaserWriter 8 driver to select a PPD, even when no printer is available on the network. We supply an option in our program to patch the user's Laserwriter driver, which allows LaserWriter 8.0 to perform the setup with no printer.

Do the LaserWriter 8 drivers provide any way to do this without patching the driver, and will this patch work on later LaserWriter 8 drivers, in particular 8.1.1, 8.2, and 8.2.2?

A: At present, there is no way to do this. Many customers have requested a feature that would allow them to either set up a printer or select a PPD file when there is no device visible in the chooser. This feature is particularly attractive to PowerBook users, and it is something we expect to add in a future version of the driver.

[Jul 01 1995]
