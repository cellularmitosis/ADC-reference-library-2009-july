New Features For Creating More Interactive Movies

QuickTime 4.1 introduces a set of new features that allow for the creation of more advanced, interactive movies. These features include

  • the introduction of embedded movies

  • addition of new wired actions and events

  • addition of new ways to communicate between a wired movie and JavaScript in a Web browser

For more information about wired movies in QuickTime 4, refer to QuickTime Wired Movies And Sprite Animation, which describes and documents the API for creating interactive movies and sprites, including HREF Tracks and wired sprites. This is available for download as a PDF at

<http://developer.apple.com/documentation/quicktime/qtdevdocs/RM/PDF.htm>

For sample code that enables you to use wired movies, refer to

<http://www.apple.com/quicktime/developers/samplecode.html>
