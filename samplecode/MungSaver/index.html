MungSaver

Posted: 2003-11-18

Description

This sample is a simple Screen Saver module for Mac OS X 10.3 which uses the Sequence Grabber DataProc mung technique demonstrated in samples such as MungGrab (part of the SGDataProc Sample). This sample uses the vImage library to perform some basic pixel munging on the source image and can be used with any FireWire or USB camera. Place MungSaver.saver is your /Library/Screen Savers/ folder then open System Preferences and choose the module from the "Desktop & Screen Savers" preference panel. Make sure a camera is plugged in and turned on. Requirements: Xcode 1.0 or later, Mac OS X 10.3 (Panther) or later with QuickTime 6.4 or later. Keywords: Sequence Grabber, Capture
