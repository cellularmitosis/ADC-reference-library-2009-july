Description

CFPrefsDumper is a project to show how to use the CFPreferences API's to access information about the current user preferences. Requirements: Mac OS X, Project Builder (CodeWarrior 6,7 or 8 for CFM projects) We hope this helps people get up and running with HID Manager in a quick and painless manner.
