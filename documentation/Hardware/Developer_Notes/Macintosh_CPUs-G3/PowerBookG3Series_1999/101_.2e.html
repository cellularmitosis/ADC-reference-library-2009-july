Changing the Operation of the Keyboard

Several of the keys on the keyboard have more than one mode of operation. The function keys can also control the display and speakers; the keys on the right side of the keyboard can also be used as a numeric keypad; and certain control keys can also be used as page-control keys. These changes are controlled by the Fn key, the Num Lock key, and the Function Keys checkbox in the Keyboard control panel.
